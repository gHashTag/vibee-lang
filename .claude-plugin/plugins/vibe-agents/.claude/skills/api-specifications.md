## Описание

Создание и поддержка API спецификаций для проекта Vibee. OpenAPI, JSON Schema, документация API, валидация запросов.

## Основные возможности

- **OpenAPI спецификации**: Создание и поддержка OpenAPI 3.0 спецификаций
- **JSON Schema**: Определение схем данных
- **Генерация документации**: Автоматическая генерация документации из спецификаций
- **Валидация**: Runtime валидация API запросов/ответов
- **Mock серверы**: Создание mock серверов из спецификаций

## Примеры использования

- Создание OpenAPI спецификации для REST API
- Документирование MCP серверов
- Валидация запросов к внешним API
- Генерация TypeScript типов из OpenAPI схем

## Связанные скиллы

- `typescript-type-safety` - генерация типов из схем
- `documentation-knowledge` - документация API
- `mcp-integration` - спецификации MCP серверов

## Best Practices

- Всегда документируй API через спецификации
- Используй JSON Schema для валидации данных
- Генерируй типы из спецификаций для type safety
- Поддерживай версионирование API
- Включай примеры запросов/ответов в документацию
