## Описание

Настройка и поддержка CI/CD пайплайнов для проекта Vibee. Автоматизация тестирования, сборки и развертывания.

## Основные возможности

- **Автоматическое тестирование**: Запуск тестов при каждом коммите
- **Проверка типов**: TypeScript type checking в пайплайне
- **Сборка**: Автоматическая сборка проекта
- **Деплой**: Автоматическое развертывание на staging/production
- **Уведомления**: Интеграция с уведомлениями об успехе/ошибках

## Примеры использования

- Настройка GitHub Actions для автоматического тестирования
- Создание пайплайна для деплоя в Kubernetes
- Настройка автоматического запуска Rainbow Bridge тестов
- Интеграция с Sentry для отслеживания деплоев

## Связанные скиллы

- `testing-tdd` - тесты в пайплайне
- `devops-deployment` - развертывание
- `typescript-type-safety` - проверка типов

## Best Practices

- Всегда запускай проверку типов перед тестами
- Используй кэширование зависимостей для ускорения
- Разделяй пайплайны на этапы (lint → typecheck → test → build → deploy)
- Настраивай rollback стратегию для production деплоев
- Логируй все этапы пайплайна для отладки
