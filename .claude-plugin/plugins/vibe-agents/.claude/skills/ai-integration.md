## Описание

Скилл для работы с AI и LLM интеграциями в проекте Vibee. Включает настройку провайдеров, оптимизацию промптов, управление токенами и интеграцию с различными LLM API.

## Основные возможности

- **Интеграция провайдеров**: OpenAI, Anthropic, OpenRouter, локальные модели
- **Управление промптами**: Chain-of-Thought, ReAct, Few-shot, Zero-shot техники
- **Оптимизация затрат**: Отслеживание токенов, выбор оптимальных моделей
- **Embeddings**: Работа с векторными представлениями для поиска и RAG
- **Fine-tuning**: Настройка моделей под специфические задачи

## Примеры использования

- Настройка нового LLM провайдера в ElizaOS
- Оптимизация промптов для лучшего качества ответов
- Интеграция с OpenRouter для доступа к множеству моделей
- Настройка системы промптов для VibeMates агентов

## Связанные скиллы

- `langfuse-observability` - отслеживание использования LLM
- `elizaos-plugin` - создание плагинов с AI функциональностью
- `mcp-integration` - интеграция через Model Context Protocol

## Best Practices

- Всегда используй Langfuse для трейсинга LLM вызовов
- Оптимизируй промпты для минимизации токенов
- Используй кэширование для повторяющихся запросов
- Настрой fallback стратегии при недоступности провайдера
