## Описание

Написание кода и реализация функциональности в проекте Vibee. Следование принципам функционального программирования, TDD подходу, создание чистого и поддерживаемого кода.

## Основные возможности

- **Функциональное программирование**: Использование pure functions, композиции, монад
- **TDD подход**: Написание тестов перед кодом
- **Чистый код**: Следование принципам SOLID и чистого кода
- **Модульность**: Создание маленьких, переиспользуемых модулей
- **Type Safety**: Использование строгой типизации TypeScript

## Примеры использования

- Реализация нового функционала по требованиям
- Рефакторинг существующего кода
- Создание утилитных функций
- Реализация бизнес-логики плагинов

## Связанные скиллы

- `testing-tdd` - написание тестов
- `typescript-type-safety` - типизация кода
- `code-review` - проверка качества

## Best Practices

- Всегда следуй TDD циклу (тест → код → рефакторинг)
- Используй функциональное программирование (pure functions, композиция)
- Создавай маленькие, сфокусированные функции
- Проверяй типы после каждого изменения (`bun run typecheck`)
- Документируй сложную логику
- Избегай side effects где возможно
