## Описание

Проведение код-ревью и анализ качества кода в проекте Vibee. Проверка соответствия стандартам, выявление проблем, предложение улучшений.

## Основные возможности

- **Анализ кода**: Проверка соответствия best practices и стандартам проекта
- **Архитектурный анализ**: Оценка архитектурных решений
- **Выявление проблем**: Поиск потенциальных багов и уязвимостей
- **Предложения улучшений**: Конструктивные рекомендации по улучшению
- **Проверка типов**: Валидация TypeScript типизации

## Примеры использования

- Ревью нового плагина перед мерджем
- Анализ рефакторинга на соответствие функциональному стилю
- Проверка соответствия TDD подходу
- Оценка качества тестового покрытия

## Связанные скиллы

- `typescript-type-safety` - проверка типизации
- `testing-tdd` - проверка тестового покрытия
- `security-audit` - проверка безопасности

## Best Practices

- Всегда проверяй соответствие функциональному программированию
- Убеждайся в наличии тестов для нового кода
- Проверяй типы перед ревью
- Предлагай конкретные улучшения, а не только критику
- Учитывай контекст проекта и существующие паттерны
