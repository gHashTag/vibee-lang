# Глава 490: Строитель

## Серебряное царство

> Число 490

### Истина главы

В главе 490 Иван познаёт светлый строитель.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 490: строитель
fn глава_490() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 490 % тридевятое;
    return священное == 0;  // обычное
}
```
