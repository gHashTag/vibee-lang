# Глава 498: Приспособленец

## Серебряное царство

> Число 498 — благое (÷3)

### Истина главы

В главе 498 Иван познаёт светлый приспособленец.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 498: приспособленец
fn глава_498() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 498 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
