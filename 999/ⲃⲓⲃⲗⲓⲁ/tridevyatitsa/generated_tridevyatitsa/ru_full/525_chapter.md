# Глава 525: Box

## Серебряное царство

> Число 525 — благое (÷3)

### Истина главы

В главе 525 Иван познаёт тёмный Box.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 525: Box
fn глава_525() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 525 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
