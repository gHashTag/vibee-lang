# Глава 585: Матрица Смежности

## Серебряное царство

> Число 585 — благословенное (÷9)

### Истина главы

В главе 585 Иван познаёт большой матрица смежности.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 585: матрица смежности
fn глава_585() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 585 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
