# Глава 198: Правь

## Медное царство

> Число 198 — благословенное (÷9)

### Истина главы

В главе 198 Иван познаёт большой Правь.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 198: Правь
fn глава_198() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 198 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

> Правь есть большой истина.
