# Глава 789: Биокомпьютеры

## Золотое царство

> Число 789 — благое (÷3)

### Истина главы

В главе 789 Иван познаёт светлый биокомпьютеры.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 789: биокомпьютеры
fn глава_789() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 789 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Биокомпьютеры есть светлый истина.
> Познать — путь героя.
