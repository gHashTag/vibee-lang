# Глава 243: Книга Судеб

## Медное царство

> Число 243 — священное (÷27)

### Истина главы

В главе 243 Иван познаёт тёмный книга судеб.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 243: книга судеб
fn глава_243() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 243 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
