# Глава 906: Distributed Transaction

## Золотое царство

> Число 906 — благое (÷3)

### Истина главы

В главе 906 Иван познаёт большой distributed transaction.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 906: distributed transaction
fn глава_906() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 906 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
