# Глава 288: Речь

## Медное царство

> Число 288 — благословенное (÷9)

### Истина главы

В главе 288 Иван познаёт святой речь.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 288: речь
fn глава_288() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 288 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
