# Глава 510: Интерпретатор

## Серебряное царство

> Число 510 — благое (÷3)

### Истина главы

В главе 510 Иван познаёт тёмный интерпретатор.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 510: интерпретатор
fn глава_510() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 510 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Интерпретатор есть тёмный истина.
> Соединить — путь героя.
