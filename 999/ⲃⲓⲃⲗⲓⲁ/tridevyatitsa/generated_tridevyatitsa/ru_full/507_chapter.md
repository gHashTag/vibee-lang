# Глава 507: Стратегия

## Серебряное царство

> Число 507 — благое (÷3)

### Истина главы

В главе 507 Иван познаёт светлый стратегия.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 507: стратегия
fn глава_507() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 507 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
