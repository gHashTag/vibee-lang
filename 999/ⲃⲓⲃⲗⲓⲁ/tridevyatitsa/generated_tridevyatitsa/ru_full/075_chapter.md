# Глава 75: Небесное Царство

## Медное царство

> Число 75 — благое (÷3)

### Истина главы

В главе 75 Иван познаёт светлый небесное царство.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 75: небесное царство
fn глава_75() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 75 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
