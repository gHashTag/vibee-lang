# Глава 192: Буян-Остров

## Медное царство

> Число 192 — благое (÷3)

### Истина главы

В главе 192 Иван познаёт вечный Буян-остров.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 192: Буян-остров
fn глава_192() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 192 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
