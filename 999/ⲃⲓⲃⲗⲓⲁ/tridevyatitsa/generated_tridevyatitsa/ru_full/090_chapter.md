# Глава 90: Зеркальце Волшебное

## Медное царство

> Число 90 — благословенное (÷9)

### Истина главы

В главе 90 Иван познаёт мудрый зеркальце волшебное.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 90: зеркальце волшебное
fn глава_90() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 90 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
