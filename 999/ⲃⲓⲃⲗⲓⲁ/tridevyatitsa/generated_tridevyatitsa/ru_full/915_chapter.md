# Глава 915: Crdts

## Золотое царство

> Число 915 — благое (÷3)

### Истина главы

В главе 915 Иван познаёт добрый CRDTs.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 915: CRDTs
fn глава_915() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 915 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
