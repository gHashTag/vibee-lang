# Глава 120: Месть

## Медное царство

> Число 120 — благое (÷3)

### Истина главы

В главе 120 Иван познаёт тёмный месть.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 120: месть
fn глава_120() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 120 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Месть есть тёмный истина.
