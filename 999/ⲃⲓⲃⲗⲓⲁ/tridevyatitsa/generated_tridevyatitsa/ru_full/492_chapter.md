# Глава 492: Одиночка

## Серебряное царство

> Число 492 — благое (÷3)

### Истина главы

В главе 492 Иван познаёт малый одиночка.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 492: одиночка
fn глава_492() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 492 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Одиночка есть малый истина.
> Разделить — путь героя.
