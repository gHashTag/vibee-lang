# Глава 405: Thunk

## Серебряное царство

> Число 405 — священное (÷27)

### Истина главы

В главе 405 Иван познаёт большой Thunk.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 405: Thunk
fn глава_405() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 405 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

### Мудрость

> Thunk есть большой истина.
> Разрушить — путь героя.
