# Глава 564: Сортировка Блинами

## Серебряное царство

> Число 564 — благое (÷3)

### Истина главы

В главе 564 Иван познаёт святой сортировка блинами.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 564: сортировка блинами
fn глава_564() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 564 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
