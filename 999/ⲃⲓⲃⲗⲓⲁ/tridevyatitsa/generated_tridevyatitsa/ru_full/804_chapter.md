# Глава 804: Червоточины

## Золотое царство

> Число 804 — благое (÷3)

### Истина главы

В главе 804 Иван познаёт светлый червоточины.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 804: червоточины
fn глава_804() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 804 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
