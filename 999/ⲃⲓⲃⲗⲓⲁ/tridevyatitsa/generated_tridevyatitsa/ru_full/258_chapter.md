# Глава 258: Храбрость

## Медное царство

> Число 258 — благое (÷3)

### Истина главы

В главе 258 Иван познаёт малый храбрость.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 258: храбрость
fn глава_258() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 258 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Храбрость есть малый истина.
> Разрушить — путь героя.
> Делай добро.
