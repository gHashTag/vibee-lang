# Глава 483: Поиск Цикла

## Серебряное царство

> Число 483 — благое (÷3)

### Истина главы

В главе 483 Иван познаёт тёмный поиск цикла.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 483: поиск цикла
fn глава_483() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 483 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
