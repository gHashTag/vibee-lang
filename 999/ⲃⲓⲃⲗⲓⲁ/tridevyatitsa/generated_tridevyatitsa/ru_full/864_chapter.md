# Глава 864: Отладка

## Золотое царство

> Число 864 — священное (÷27)

### Истина главы

В главе 864 Иван познаёт святой отладка.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 864: отладка
fn глава_864() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 864 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
