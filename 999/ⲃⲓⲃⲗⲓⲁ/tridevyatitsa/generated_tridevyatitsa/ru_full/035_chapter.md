# Глава 35: Конёк-Горбунок

## Медное царство

> Число 35

### Истина главы

В главе 35 Иван познаёт тёмный Конёк-Горбунок.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 35: Конёк-Горбунок
fn глава_35() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 35 % тридевятое;
    return священное == 0;  // обычное
}
```
