# Глава 95: Живая Вода

## Медное царство

> Число 95

### Истина главы

В главе 95 Иван познаёт тёмный живая вода.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 95: живая вода
fn глава_95() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 95 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Живая Вода есть тёмный истина.
> Разрушить — путь героя.
> Делай добро.
