# Глава 180: Морозко

## Медное царство

> Число 180 — благословенное (÷9)

### Истина главы

В главе 180 Иван познаёт добрый Морозко.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 180: Морозко
fn глава_180() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 180 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
