# Глава 195: Река Смородина

## Медное царство

> Число 195 — благое (÷3)

### Истина главы

В главе 195 Иван познаёт живой река Смородина.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 195: река Смородина
fn глава_195() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 195 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
