# Глава 408: Быстрая Сортировка

## Серебряное царство

> Число 408 — благое (÷3)

### Истина главы

В главе 408 Иван познаёт светлый быстрая сортировка.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 408: быстрая сортировка
fn глава_408() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 408 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Быстрая Сортировка есть светлый истина.
> Соединить — путь героя.
