# Глава 624: Абстрактная Фабрика

## Серебряное царство

> Число 624 — благое (÷3)

### Истина главы

В главе 624 Иван познаёт мудрый абстрактная фабрика.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: мудрый конец

```vibee
// Глава 624: абстрактная фабрика
fn глава_624() {

    let тридевятое = тройка * тройка * тройка;  // 27

}
```
