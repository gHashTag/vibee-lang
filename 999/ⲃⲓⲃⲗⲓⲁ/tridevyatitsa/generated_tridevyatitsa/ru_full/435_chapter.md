# Глава 435: Двусвязный Список

## Серебряное царство

> Число 435 — благое (÷3)

### Истина главы

В главе 435 Иван познаёт мудрый двусвязный список.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 435: двусвязный список
fn глава_435() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 435 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
