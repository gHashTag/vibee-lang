# Глава 127: Добро

## Медное царство

> Число 127

### Истина главы

В главе 127 Иван познаёт большой добро.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 127: добро
fn глава_127() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 127 % тридевятое;
    return священное == 0;  // обычное
}
```
