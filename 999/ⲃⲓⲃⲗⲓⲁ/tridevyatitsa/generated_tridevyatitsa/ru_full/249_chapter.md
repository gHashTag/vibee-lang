# Глава 249: Совесть

## Медное царство

> Число 249 — благое (÷3)

### Истина главы

В главе 249 Иван познаёт святой совесть.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 249: совесть
fn глава_249() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 249 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Совесть есть святой истина.
> Соединить — путь героя.
