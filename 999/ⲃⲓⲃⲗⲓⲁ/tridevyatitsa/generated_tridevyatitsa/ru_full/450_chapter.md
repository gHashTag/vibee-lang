# Глава 450: Матрица Смежности

## Серебряное царство

> Число 450 — благословенное (÷9)

### Истина главы

В главе 450 Иван познаёт малый матрица смежности.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 450: матрица смежности
fn глава_450() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 450 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
