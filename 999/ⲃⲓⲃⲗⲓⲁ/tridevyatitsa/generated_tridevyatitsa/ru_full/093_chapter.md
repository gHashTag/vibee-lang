# Глава 93: Яйцо Кощея

## Медное царство

> Число 93 — благое (÷3)

### Истина главы

В главе 93 Иван познаёт светлый яйцо Кощея.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 93: яйцо Кощея
fn глава_93() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 93 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Яйцо Кощея есть светлый истина.
