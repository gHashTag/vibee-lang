# Глава 63: Правь

## Медное царство

> Число 63 — благословенное (÷9)

### Истина главы

В главе 63 Иван познаёт светлый Правь.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 63: Правь
fn глава_63() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 63 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
