# Глава 381: Комплексное Число

## Серебряное царство

> Число 381 — благое (÷3)

### Истина главы

В главе 381 Иван познаёт светлый комплексное число.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 381: комплексное число
fn глава_381() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 381 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
