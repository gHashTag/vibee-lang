# Глава 351: Поиск Точек Сочленения

## Серебряное царство

> Число 351 — священное (÷27)

### Истина главы

В главе 351 Иван познаёт вечный поиск точек сочленения.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 351: поиск точек сочленения
fn глава_351() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 351 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
