# Глава 114: Совесть

## Медное царство

> Число 114 — благое (÷3)

### Истина главы

В главе 114 Иван познаёт мудрый совесть.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 114: совесть
fn глава_114() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 114 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
