# Глава 429: Сортировка Блинами

## Серебряное царство

> Число 429 — благое (÷3)

### Истина главы

В главе 429 Иван познаёт малый сортировка блинами.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 429: сортировка блинами
fn глава_429() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 429 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
