# Глава 363: Приспособленец

## Серебряное царство

> Число 363 — благое (÷3)

### Истина главы

В главе 363 Иван познаёт добрый приспособленец.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 363: приспособленец
fn глава_363() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 363 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
