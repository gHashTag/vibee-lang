# Глава 750: Dht

## Золотое царство

> Число 750 — благое (÷3)

### Истина главы

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 750: DHT
fn глава_750() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 750 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
