# Глава 768: Isolation

## Золотое царство

> Число 768 — благое (÷3)

### Истина главы

В главе 768 Иван познаёт малый isolation.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 768: isolation
fn глава_768() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 768 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
