# Глава 795: Диффузионные Модели

## Золотое царство

### Истина главы

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 795: диффузионные модели
fn глава_795() {

    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 795 % тридевятое;

}
```
