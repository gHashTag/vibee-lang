# Глава 432: Квантовая Сортировка

## Серебряное царство

> Число 432 — священное (÷27)

### Истина главы

В главе 432 Иван познаёт светлый квантовая сортировка.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 432: квантовая сортировка
fn глава_432() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 432 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
