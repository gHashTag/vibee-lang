# Глава 7: Гармония

## Медное царство

> Число 7

### Истина главы

В главе 7 Иван познаёт вечный гармония.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 7: гармония
fn глава_7() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 7 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Гармония есть вечный истина.
> Разрушить — путь героя.
