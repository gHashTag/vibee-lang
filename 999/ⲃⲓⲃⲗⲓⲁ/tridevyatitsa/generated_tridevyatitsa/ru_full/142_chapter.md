# Глава 142: Гармония

## Медное царство

> Число 142

### Истина главы

В главе 142 Иван познаёт светлый гармония.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 142: гармония

    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 142 % тридевятое;
    return священное == 0;  // обычное
}
```
