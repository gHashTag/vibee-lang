# Глава 327: Буян-Остров

## Медное царство

> Число 327 — благое (÷3)

### Истина главы

В главе 327 Иван познаёт большой Буян-остров.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 327: Буян-остров
fn глава_327() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 327 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
