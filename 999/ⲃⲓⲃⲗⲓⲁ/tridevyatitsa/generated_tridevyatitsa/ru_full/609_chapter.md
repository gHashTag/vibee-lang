# Глава 609: Суффиксный Массив

## Серебряное царство

> Число 609 — благое (÷3)

### Истина главы

В главе 609 Иван познаёт святой суффиксный массив.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 609: суффиксный массив
fn глава_609() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 609 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Суффиксный Массив есть святой истина.
> Познать — путь героя.
