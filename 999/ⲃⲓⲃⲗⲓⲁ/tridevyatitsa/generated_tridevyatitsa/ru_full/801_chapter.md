# Глава 801: Терраформирование

## Золотое царство

> Число 801 — благословенное (÷9)

### Истина главы

В главе 801 Иван познаёт малый терраформирование.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 801: терраформирование
fn глава_801() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 801 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
