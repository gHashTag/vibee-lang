# Глава 216: Живая Вода

## Медное царство

> Число 216 — священное (÷27)

### Истина главы

В главе 216 Иван познаёт мудрый живая вода.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 216: живая вода
fn глава_216() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 216 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

### Мудрость

> Живая Вода есть мудрый истина.
> Разрушить — путь героя.
> Делай добро.
