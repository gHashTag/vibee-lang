# Глава 651: Комплексное Число

## Серебряное царство

> Число 651 — благое (÷3)

### Истина главы

В главе 651 Иван познаёт большой комплексное число.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 651: комплексное число
fn глава_651() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 651 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Комплексное Число есть большой истина.
> Разрушить — путь героя.
> Делай добро.
