# Глава 291: Высота

## Медное царство

> Число 291 — благое (÷3)

### Истина главы

В главе 291 Иван познаёт светлый высота.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 291: высота
fn глава_291() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 291 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Высота есть светлый истина.
> Создать — путь героя.
> Делай добро.
