# Глава 783: Proof Of Correctness

## Золотое царство

> Число 783 — священное (÷27)

### Истина главы

В главе 783 Иван познаёт тёмный proof of correctness.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 783: proof of correctness
fn глава_783() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27

    return священное == 0;  // священное (÷27)
}
```
