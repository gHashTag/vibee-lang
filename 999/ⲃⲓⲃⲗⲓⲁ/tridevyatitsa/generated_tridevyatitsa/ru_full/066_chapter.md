# Глава 66: Синее Море

## Медное царство

> Число 66 — благое (÷3)

### Истина главы

В главе 66 Иван познаёт малый синее море.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 66: синее море
fn глава_66() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 66 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Синее Море есть малый истина.
> Соединить — путь героя.
> Делай добро.
