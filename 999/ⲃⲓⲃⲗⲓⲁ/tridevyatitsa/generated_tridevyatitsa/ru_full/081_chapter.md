# Глава 81: Живая Вода

## Медное царство

> Число 81 — священное (÷27)

### Истина главы

В главе 81 Иван познаёт тёмный живая вода.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 81: живая вода
fn глава_81() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 81 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

### Мудрость

> Живая Вода есть тёмный истина.
> Скрыть — путь героя.
> Делай добро.
