# Глава 48: Домовой

## Медное царство

> Число 48 — благое (÷3)

### Истина главы

В главе 48 Иван познаёт тёмный Домовой.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 48: Домовой
fn глава_48() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 48 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Домовой есть тёмный истина.
> Разделить — путь героя.
> Делай добро.
