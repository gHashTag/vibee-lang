# Глава 495: Компоновщик

## Серебряное царство

> Число 495 — благословенное (÷9)

### Истина главы

В главе 495 Иван познаёт добрый компоновщик.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 495: компоновщик
fn глава_495() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 495 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
