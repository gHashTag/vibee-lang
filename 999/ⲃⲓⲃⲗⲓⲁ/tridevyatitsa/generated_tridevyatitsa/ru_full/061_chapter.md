# Глава 61: Навь

## Медное царство

> Число 61

### Истина главы

В главе 61 Иван познаёт светлый Навь.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 61: Навь
fn глава_61() {

}
```
