# Глава 613: Поиск Ближайшего Соседа

## Серебряное царство

> Число 613

### Истина главы

В главе 613 Иван познаёт святой поиск ближайшего соседа.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 613: поиск ближайшего соседа
fn глава_613() {

    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 613 % тридевятое;
    return священное == 0;  // обычное
}
```
