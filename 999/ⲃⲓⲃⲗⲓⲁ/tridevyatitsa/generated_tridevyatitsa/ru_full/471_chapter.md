# Глава 471: Бойер-Мур

## Серебряное царство

> Число 471 — благое (÷3)

### Истина главы

В главе 471 Иван познаёт малый Бойер-Мур.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: малый конец

### Код

// Глава 471: Бойер-Мур

}
