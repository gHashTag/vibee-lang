# Глава 765: Abac

## Золотое царство

> Число 765 — благословенное (÷9)

### Истина главы

В главе 765 Иван познаёт большой ABAC.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 765: ABAC
fn глава_765() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 765 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
