# Глава 543: Быстрая Сортировка

## Серебряное царство

> Число 543 — благое (÷3)

### Истина главы

В главе 543 Иван познаёт добрый быстрая сортировка.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 543: быстрая сортировка
fn глава_543() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 543 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
