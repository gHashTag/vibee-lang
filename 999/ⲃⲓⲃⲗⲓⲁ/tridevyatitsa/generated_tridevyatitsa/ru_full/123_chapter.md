# Глава 123: Храбрость

## Медное царство

> Число 123 — благое (÷3)

### Истина главы

В главе 123 Иван познаёт большой храбрость.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 123: храбрость
fn глава_123() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 123 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Храбрость есть большой истина.
> Соединить — путь героя.
> Делай добро.
