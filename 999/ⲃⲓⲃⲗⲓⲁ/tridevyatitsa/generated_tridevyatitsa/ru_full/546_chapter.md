# Глава 546: Сортировка Подсчётом

## Серебряное царство

> Число 546 — благое (÷3)

### Истина главы

В главе 546 Иван познаёт большой сортировка подсчётом.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 546: сортировка подсчётом
fn глава_546() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 546 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Сортировка Подсчётом есть большой истина.
> Соединить — путь героя.
