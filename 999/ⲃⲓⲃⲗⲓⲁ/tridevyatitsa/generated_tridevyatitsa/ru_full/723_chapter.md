# Глава 723: Каналы

## Золотое царство

> Число 723 — благое (÷3)

### Истина главы

В главе 723 Иван познаёт тёмный каналы.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 723: каналы
fn глава_723() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 723 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Каналы есть тёмный истина.
> Разрушить — путь героя.
> Делай добро.
