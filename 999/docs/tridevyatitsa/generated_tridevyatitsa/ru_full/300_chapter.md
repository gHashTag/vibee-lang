# Глава 300: Баба-Яга

## Медное царство

> Число 300 — благое (÷3)

### Истина главы

В главе 300 Иван познаёт вечный Баба-Яга.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 300: Баба-Яга
fn глава_300() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 300 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Баба-Яга есть вечный истина.
> Разделить — путь героя.
> Делай добро.
