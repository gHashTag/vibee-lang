# Глава 552: Гномья Сортировка

## Серебряное царство

> Число 552 — благое (÷3)

### Истина главы

В главе 552 Иван познаёт большой гномья сортировка.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 552: гномья сортировка
fn глава_552() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 552 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
