# Глава 18: Речь

## Медное царство

> Число 18 — благословенное (÷9)

### Истина главы

В главе 18 Иван познаёт добрый речь.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 18: речь
fn глава_18() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 18 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

> Речь есть добрый истина.
> Создать — путь героя.
