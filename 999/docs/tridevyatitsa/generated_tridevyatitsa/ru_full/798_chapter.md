# Глава 798: Технологическая Сингулярность

## Золотое царство

> Число 798 — благое (÷3)

### Истина главы

В главе 798 Иван познаёт живой технологическая сингулярность.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 798: технологическая сингулярность
fn глава_798() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 798 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
