# Глава 171: Царевна-Лягушка

## Медное царство

> Число 171 — благословенное (÷9)

### Истина главы

В главе 171 Иван познаёт тёмный Царевна-лягушка.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 171: Царевна-лягушка
fn глава_171() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 171 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
