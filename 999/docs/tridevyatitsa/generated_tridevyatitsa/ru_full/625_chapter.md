# Глава 625: Строитель

## Серебряное царство

> Число 625

### Истина главы

В главе 625 Иван познаёт живой строитель.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: живой конец

### Код

// Глава 625: строитель
fn глава_625() {

}
