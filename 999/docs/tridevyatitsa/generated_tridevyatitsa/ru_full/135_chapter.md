# Глава 135: Бессмертие

## Медное царство

> Число 135 — священное (÷27)

### Истина главы

В главе 135 Иван познаёт вечный бессмертие.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 135: бессмертие
fn глава_135() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 135 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

### Мудрость

> Бессмертие есть вечный истина.
> Соединить — путь героя.
> Делай добро.
