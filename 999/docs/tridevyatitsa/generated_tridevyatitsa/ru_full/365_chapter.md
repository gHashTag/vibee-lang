# Глава 365: Цепочка Обязанностей

## Серебряное царство

> Число 365

### Истина главы

В главе 365 Иван познаёт святой цепочка обязанностей.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 365: цепочка обязанностей
fn глава_365() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 365 % тридевятое;
    return священное == 0;  // обычное
}
```
