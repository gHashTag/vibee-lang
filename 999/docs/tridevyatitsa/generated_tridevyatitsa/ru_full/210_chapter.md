# Глава 210: Небесное Царство

## Медное царство

> Число 210 — благое (÷3)

### Истина главы

В главе 210 Иван познаёт живой небесное царство.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 210: небесное царство
fn глава_210() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 210 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Небесное Царство есть живой истина.
> Познать — путь героя.
