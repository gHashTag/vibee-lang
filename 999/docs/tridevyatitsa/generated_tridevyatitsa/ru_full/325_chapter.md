# Глава 325: Тридевятое Царство

## Медное царство

> Число 325

### Истина главы

В главе 325 Иван познаёт тёмный Тридевятое царство.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 325: Тридевятое царство
fn глава_325() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 325 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Тридевятое Царство есть тёмный истина.
> Разрушить — путь героя.
> Делай добро.
