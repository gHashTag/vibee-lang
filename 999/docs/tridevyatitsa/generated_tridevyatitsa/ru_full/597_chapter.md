# Глава 597: Интерполяционный Поиск

## Серебряное царство

> Число 597 — благое (÷3)

### Истина главы

В главе 597 Иван познаёт малый интерполяционный поиск.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 597: интерполяционный поиск
fn глава_597() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 597 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
