# Глава 581: Куча

## Серебряное царство

> Число 581

### Истина главы

В главе 581 Иван познаёт большой куча.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 581: куча
fn глава_581() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 581 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Куча есть большой истина.
> Разрушить — путь героя.
> Делай добро.
