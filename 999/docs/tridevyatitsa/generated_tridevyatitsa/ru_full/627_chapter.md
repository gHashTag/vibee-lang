# Глава 627: Одиночка

## Серебряное царство

> Число 627 — благое (÷3)

### Истина главы

В главе 627 Иван познаёт святой одиночка.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 627: одиночка
fn глава_627() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 627 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
