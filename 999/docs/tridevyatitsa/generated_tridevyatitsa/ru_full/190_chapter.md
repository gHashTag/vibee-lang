# Глава 190: Тридевятое Царство

## Медное царство

> Число 190

### Истина главы

В главе 190 Иван познаёт малый Тридевятое царство.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 190: Тридевятое царство
fn глава_190() {

    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 190 % тридевятое;
    return священное == 0;  // обычное
}
```
