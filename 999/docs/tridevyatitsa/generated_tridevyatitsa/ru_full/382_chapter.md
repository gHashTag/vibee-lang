# Глава 382: Рациональное Число

## Серебряное царство

> Число 382

### Истина главы

В главе 382 Иван познаёт большой рациональное число.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 382: рациональное число
fn глава_382() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 382 % тридевятое;
    return священное == 0;  // обычное
}
```
