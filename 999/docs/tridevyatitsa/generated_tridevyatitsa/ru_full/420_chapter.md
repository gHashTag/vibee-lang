# Глава 420: Сортировка Деревом

## Серебряное царство

> Число 420 — благое (÷3)

### Истина главы

В главе 420 Иван познаёт большой сортировка деревом.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 420: сортировка деревом
fn глава_420() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 420 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Сортировка Деревом есть большой истина.
> Разделить — путь героя.
