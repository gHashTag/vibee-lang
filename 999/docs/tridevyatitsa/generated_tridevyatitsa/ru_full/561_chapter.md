# Глава 561: Чётно-Нечётная Сортировка

## Серебряное царство

> Число 561 — благое (÷3)

### Истина главы

В главе 561 Иван познаёт тёмный чётно-нечётная сортировка.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 561: чётно-нечётная сортировка
fn глава_561() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 561 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
