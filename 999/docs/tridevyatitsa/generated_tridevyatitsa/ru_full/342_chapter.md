# Глава 342: Поиск В Дереве

## Серебряное царство

> Число 342 — благословенное (÷9)

### Истина главы

В главе 342 Иван познаёт малый поиск в дереве.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 342: поиск в дереве
fn глава_342() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 342 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
