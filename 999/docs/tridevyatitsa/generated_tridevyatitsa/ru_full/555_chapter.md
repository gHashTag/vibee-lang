# Глава 555: Сортировка Деревом

## Серебряное царство

> Число 555 — благое (÷3)

### Истина главы

В главе 555 Иван познаёт мудрый сортировка деревом.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 555: сортировка деревом
fn глава_555() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 555 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Сортировка Деревом есть мудрый истина.
