# Глава 550: Сортировка Шелла

## Серебряное царство

> Число 550

### Истина главы

В главе 550 Иван познаёт светлый сортировка Шелла.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 550: сортировка Шелла
fn глава_550() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 550 % тридевятое;
    return священное == 0;  // обычное
}
```
