# Глава 402: Generator

## Серебряное царство

> Число 402 — благое (÷3)

### Истина главы

В главе 402 Иван познаёт светлый Generator.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 402: Generator
fn глава_402() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 402 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
