# Глава 108: Книга Судеб

## Медное царство

> Число 108 — священное (÷27)

### Истина главы

В главе 108 Иван познаёт добрый книга судеб.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 108: книга судеб
fn глава_108() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 108 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
