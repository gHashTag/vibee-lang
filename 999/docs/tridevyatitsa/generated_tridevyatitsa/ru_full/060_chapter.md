# Глава 60: Река Смородина

## Медное царство

> Число 60 — благое (÷3)

### Истина главы

В главе 60 Иван познаёт святой река Смородина.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 60: река Смородина
fn глава_60() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 60 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Река Смородина есть святой истина.
> Соединить — путь героя.
