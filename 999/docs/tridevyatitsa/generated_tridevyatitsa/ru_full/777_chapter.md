# Глава 777: Honeypot

## Золотое царство

> Число 777 — благое (÷3)

### Истина главы

В главе 777 Иван познаёт мудрый honeypot.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 777: honeypot
fn глава_777() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 777 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
