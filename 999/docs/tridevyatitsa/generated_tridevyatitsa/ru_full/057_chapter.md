# Глава 57: Буян-Остров

## Медное царство

> Число 57 — благое (÷3)

### Истина главы

В главе 57 Иван познаёт большой Буян-остров.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 57: Буян-остров
fn глава_57() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 57 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
