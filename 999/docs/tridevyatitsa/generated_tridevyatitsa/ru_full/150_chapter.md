# Глава 150: Порядок

## Медное царство

> Число 150 — благое (÷3)

### Истина главы

В главе 150 Иван познаёт вечный порядок.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 150: порядок
fn глава_150() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 150 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Порядок есть вечный истина.
> Найти — путь героя.
> Делай добро.
