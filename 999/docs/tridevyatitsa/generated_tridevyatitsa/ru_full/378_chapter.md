# Глава 378: Чистая Архитектура

## Серебряное царство

> Число 378 — священное (÷27)

### Истина главы

В главе 378 Иван познаёт большой чистая архитектура.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 378: чистая архитектура
fn глава_378() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 378 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
