# Глава 654: Символ

## Серебряное царство

> Число 654 — благое (÷3)

### Истина главы

В главе 654 Иван познаёт живой символ.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 654: символ
fn глава_654() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 654 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
