# Глава 64: Дремучий Лес

## Медное царство

> Число 64

### Истина главы

В главе 64 Иван познаёт светлый дремучий лес.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 64: дремучий лес
fn глава_64() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 64 % тридевятое;
    return священное == 0;  // обычное
}
```
