# Глава 336: Бойер-Мур

## Серебряное царство

> Число 336 — благое (÷3)

### Истина главы

В главе 336 Иван познаёт большой Бойер-Мур.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 336: Бойер-Мур
fn глава_336() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 336 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
