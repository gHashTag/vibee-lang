## Медное царство

> Число 34

### Истина главы

В главе 34 Иван познаёт тёмный Серый волк.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: тёмный конец

### Код

// Глава 34: Серый волк

    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 34 % тридевятое;

}
