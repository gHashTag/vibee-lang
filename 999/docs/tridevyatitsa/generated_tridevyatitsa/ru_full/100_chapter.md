# Глава 100: Аленький Цветочек

## Медное царство

> Число 100

### Истина главы

В главе 100 Иван познаёт добрый аленький цветочек.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 100: аленький цветочек
fn глава_100() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 100 % тридевятое;
    return священное == 0;  // обычное
}
```
