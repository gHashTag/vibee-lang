# Глава 152: Целостность

## Медное царство

> Число 152

### Истина главы

В главе 152 Иван познаёт живой целостность.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 152: целостность
fn глава_152() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 152 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Целостность есть живой истина.
> Разрушить — путь героя.
