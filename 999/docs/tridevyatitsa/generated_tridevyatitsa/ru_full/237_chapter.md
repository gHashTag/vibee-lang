# Глава 237: Серебряное Блюдечко

## Медное царство

> Число 237 — благое (÷3)

### Истина главы

В главе 237 Иван познаёт светлый серебряное блюдечко.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 237: серебряное блюдечко
fn глава_237() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 237 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Серебряное Блюдечко есть светлый истина.
> Разрушить — путь героя.
