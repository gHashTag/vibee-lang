# Глава 444: Префиксное Дерево

## Серебряное царство

> Число 444 — благое (÷3)

### Истина главы

В главе 444 Иван познаёт святой префиксное дерево.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 444: префиксное дерево
fn глава_444() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 444 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Префиксное Дерево есть святой истина.
