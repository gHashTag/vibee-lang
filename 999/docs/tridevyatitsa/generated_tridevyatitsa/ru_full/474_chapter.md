# Глава 474: Суффиксный Массив

## Серебряное царство

> Число 474 — благое (÷3)

### Истина главы

В главе 474 Иван познаёт малый суффиксный массив.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 474: суффиксный массив
fn глава_474() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 474 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
