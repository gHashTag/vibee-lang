# Глава 147: Свет

## Медное царство

> Число 147 — благое (÷3)

### Истина главы

В главе 147 Иван познаёт добрый свет.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 147: свет
fn глава_147() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 147 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
