# Глава 957: Диффузионные Модели

## Золотое царство

> Число 957 — благое (÷3)

### Истина главы

В главе 957 Иван познаёт светлый диффузионные модели.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 957: диффузионные модели
fn глава_957() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 957 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
