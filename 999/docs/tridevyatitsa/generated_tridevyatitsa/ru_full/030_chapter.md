# Глава 30: Баба-Яга

## Медное царство

> Число 30 — благое (÷3)

### Истина главы

В главе 30 Иван познаёт тёмный Баба-Яга.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 30: Баба-Яга
fn глава_30() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 30 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Баба-Яга есть тёмный истина.
> Соединить — путь героя.
> Делай добро.
