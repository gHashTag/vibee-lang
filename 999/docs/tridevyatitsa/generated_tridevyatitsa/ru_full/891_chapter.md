# Глава 891: Flow Control

## Золотое царство

> Число 891 — священное (÷27)

### Истина главы

В главе 891 Иван познаёт живой flow control.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 891: flow control
```
