# Глава 70: Светлый Терем

## Медное царство

> Число 70

### Истина главы

В главе 70 Иван познаёт малый светлый терем.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 70: светлый терем
fn глава_70() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 70 % тридевятое;
    return священное == 0;  // обычное
}
```
