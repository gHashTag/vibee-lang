# Глава 345: R-Дерево

## Серебряное царство

> Число 345 — благое (÷3)

### Истина главы

В главе 345 Иван познаёт святой R-дерево.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 345: R-дерево
fn глава_345() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 345 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
