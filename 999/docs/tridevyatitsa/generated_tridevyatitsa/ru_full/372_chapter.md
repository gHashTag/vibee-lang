# Глава 372: Стратегия

## Серебряное царство

> Число 372 — благое (÷3)

### Истина главы

В главе 372 Иван познаёт малый стратегия.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 372: стратегия
fn глава_372() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 372 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
