# Глава 132: Смерть

## Медное царство

> Число 132 — благое (÷3)

### Истина главы

В главе 132 Иван познаёт живой смерть.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 132: смерть
fn глава_132() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 132 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Смерть есть живой истина.
> Разрушить — путь героя.
> Делай добро.
