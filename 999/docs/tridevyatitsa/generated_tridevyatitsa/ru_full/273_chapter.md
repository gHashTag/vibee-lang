# Глава 273: Двойственность

## Медное царство

> Число 273 — благое (÷3)

### Истина главы

В главе 273 Иван познаёт мудрый двойственность.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 273: двойственность
fn глава_273() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27

}
```
