Формула главы:

```
n × 3^k × π^m = 25 × 3^2 × π^0 = 225.000000
```

Где:
- **n = 25** — номер буквы Ⲱ (Омега) в алфавите Тридевятицы
- **k = 2** — уровень царства (Золотое)
- **m = 0** — степень священности
- **π = 3.14159...** — связь конечного с бесконечным
