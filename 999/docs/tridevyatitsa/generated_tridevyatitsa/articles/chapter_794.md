Формула главы:

```
n × 3^k × π^m = 11 × 3^2 × π^0 = 99.000000
```

Где:
- **n = 11** — номер буквы Ⲕ (Каппа) в алфавите Тридевятицы
- **k = 2** — уровень царства (Золотое)
- **m = 0** — степень священности
