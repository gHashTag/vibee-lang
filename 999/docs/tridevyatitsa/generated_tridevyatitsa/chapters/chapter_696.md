## Ⲩ — Ипсилон: Высота

### Философский Смысл

Буква **Ⲩ** (Ипсилон) символизирует **высота** в системе Тридевятицы.

В контексте книги "Язык Vibee" это означает применение принципа высота к теме основы vibee.

### Числовое Значение

```
Ⲩ = 21

В троичной системе: 21 = 210 (base 3)

Священность числа 21:
  21 % 3 = 0  → делится на 3 ✓
  21 % 9 = 3  → не делится на 9
  21 % 27 = 21  → не делится на 27
```
