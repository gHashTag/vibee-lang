# Золотое сечение в дизайне алфавита 999

## Математические основы

### Золотое сечение (φ)
```
φ = (1 + √5) / 2 ≈ 1.618033988749895

Свойства:
φ² = φ + 1 ≈ 2.618
1/φ = φ - 1 ≈ 0.618
φ³ = φ² + φ ≈ 4.236
```

### Связь с троичностью
```
φ ≈ 1.618
3/φ ≈ 1.854
φ² ≈ 2.618
3 - φ ≈ 1.382

Интересно: φ² - φ = 1 (единица!)
          3 = φ + φ - 0.236 (почти два φ!)
```

### Золотой угол
```
θ = 360° / φ² ≈ 137.5°

Для троичной системы:
360° / 3 = 120° (углы треугольника)
120° × φ ≈ 194° 
120° / φ ≈ 74°
```
