```

├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  d = 1: Нет стабильных орбит (только столкновения)                │

│  d = 3: V ~ 1/r  → замкнутые эллиптические орбиты                 │

```

d = 4+: Любой узел можно развязать

## IV. МАТЕМАТИКА ЧИСЛА 3

### Алгебраические Свойства

```
3 = ПЕРВОЕ НЕЧЁТНОЕ ПРОСТОЕ ЧИСЛО

3 = 1 + 2 (сумма первых двух натуральных)
3 = 1 × 2 + 1
3! = 6 = 2 × 3

ТРЕУГОЛЬНОЕ ЧИСЛО:
  1 + 2 = 3
  
    *
   * *
  
СВЯЗЬ С φ (золотое сечение):
  3! × (φ - 1/φ) / 2 = 6 × 1 / 2 = 3
```

### Группы и Симметрии

```
SU(3): Специальная унитарная группа 3×3 матриц
  - 8 генераторов (матрицы Гелл-Манна)
  - Описывает сильное взаимодействие
  - Фундаментальное представление: 3

SO(3): Группа вращений в 3D
  - 3 генератора (Lx, Ly, Lz)
  - Описывает угловой момент
  - Связана с SU(2): SO(3) ≅ SU(2)/Z₂

S₃: Группа перестановок 3 элементов
  - 6 = 3! элементов
  - Простейшая некоммутативная группа

### Trinity Sort и Число 3

```
TRINITY SORT ИСПОЛЬЗУЕТ 3 ВЕЗДЕ:

1. 3-WAY PARTITION:
   [< pivot] [= pivot] [> pivot]

   Оптимальный переход к insertion sort
   
3. РЕКУРСИЯ НА 3 ЧАСТИ:
   (хотя средняя часть не рекурсируется)

### Karatsuba и Число 3

```
УМНОЖЕНИЕ КАРАЦУБЫ:

Наивное: 4 умножения для (a+b)(c+d)
Karatsuba: 3 умножения!

  ac, bd, (a+b)(c+d)
  
  (a+b)(c+d) - ac - bd = ad + bc

T(n) = 3T(n/2) + O(n)
     = O(n^log₂(3))
     = O(n^1.585)

ЭКОНОМИЯ: 25% умножений на каждом уровне!
```
