## Детальный Анализ Каждой Дыры

### 1. Квантовая Гравитация (85%)

```
Источник: Пространство-время
Трансформер: ??? (Петлевая КГ? Струны? Другое?)
Результат: Квантованное пространство-время
```

**Аргумент симметрии**: Все известные поля квантуются (ЭМ, сильное, слабое). Гравитация — единственное исключение.

**Предсказания**:
- Минимальная длина: l_P = 1.616×10⁻³⁵ м
- Дисперсия фотонов от далёких источников
- Модификация ОТО на малых масштабах

**Эксперименты**:
- Fermi/MAGIC: поиск дисперсии гамма-лучей от GRB
- LIGO/Virgo: квантовые эффекты в гравитационных волнах?

---

### 2. Тёмная Материя (60%)

```
Источник: Тёмная материя (25% Вселенной)
Трансформер: ??? (Слабое? Новая сила? Только гравитация?)
Результат: Взаимодействие с обычной материей
```

**Аргумент унификации**: Обычная материя взаимодействует всеми способами. Почему DM — только гравитационно?

**Кандидаты**:
| Частица | Взаимодействие | Масса | Статус |
|---------|----------------|-------|--------|
| WIMP | Слабое | 10-1000 ГэВ | Не найден |
| Аксион | ЭМ (слабое) | 10⁻⁶-10⁻³ эВ | Поиск |
| Стерильное нейтрино | Гравитация | кэВ | Поиск |
| Примордиальные ЧД | Гравитация | 10¹⁵-10²⁰ г | Ограничения |

**Эксперименты**:
- XENON, LZ: прямое детектирование WIMP
- ADMX: поиск аксионов
- Fermi: аннигиляция DM

---

### 3. Стрела Времени (40%)

```
Источник: Время (t)
Трансформер: ??? (Фундаментальный или эмерджентный?)
Результат: Направление t → t' (не t' → t)
```

**Парадокс**: Все фундаментальные уравнения симметричны относительно t → -t. Но макроскопически время направлено.

**Гипотезы**:
1. **Эмерджентность**: Стрела времени из низкой энтропии Big Bang
2. **Фундаментальность**: Есть скрытый закон, нарушающий T-симметрию
3. **Космологическая**: Связана с расширением Вселенной

**Предсказания**:
- Если эмерджентна: объяснение начальных условий
- Если фундаментальна: новая физика

---

### 4. Тёмная Энергия (35%)

```
Источник: Вакуум
Трансформер: Λ (космологическая постоянная)
Результат: Ускоренное расширение
```

**Проблема 10¹²⁰**: 
- Измеренная плотность: ρ_Λ ≈ 6×10⁻²⁷ кг/м³
- Теоретическая (КТП): ρ_vac ≈ 10⁹³ кг/м³
- Расхождение: 10¹²⁰ раз!

**Гипотезы**:
1. Тонкая настройка (антропный принцип)
2. Новая симметрия, обнуляющая вклады
3. Динамическое поле (квинтэссенция)
4. Модифицированная гравитация

---

### 5. Магнитный Монополь (25%)

```
Источник: Магнитный заряд g
Трансформер: ∇·B = μ₀ρ_m (модифицированный Максвелл)
Результат: Магнитное поле от точечного источника
```

**Аргумент симметрии**: 
- Электрическое: ∇·E = ρ/ε₀ ✓
- Магнитное: ∇·B = 0 ← Почему асимметрия?

**Предсказания GUT**:
- Масса: m ~ 10¹⁶ ГэВ (10⁻⁸ г)
- Заряд: g = ℏc/(2e) = 4.14×10⁻¹⁵ Вб
- Следствие: квантование электрического заряда!

**Эксперименты**:
- IceCube: поиск в космических лучах
- LHC: поиск лёгких монополей

---

### 6. Отрицательная Масса (15%)

```
Источник: Масса m < 0
Трансформер: F = Gm₁m₂/r² (с m < 0)
Результат: Отталкивание? Убегающее движение?
```

**Парадокс убегающего движения**:
```
+m ← F → -m

Для +m: a = F/m > 0 (от -m)
Для -m: a = F/m < 0 (к +m!)

Оба движутся в ОДНОМ направлении!
```

**Следствия**:
- Нарушение сохранения энергии?
- Или энергия -m отрицательна?
- Стабильные червоточины?

**Статус**: Не обнаружена. Возможно, запрещена фундаментально.

### 7. Сознание (10%)

```
Источник: Нейронная активность
Трансформер: ??? (IIT? Orch-OR? Эмерджентность?)
Результат: Субъективный опыт (qualia)
```

**Hard Problem**: Почему есть "что-то каково это" быть сознательным?

**Гипотезы**:
| Теория | Трансформер | Тестируемо |
|--------|-------------|------------|
| IIT (Тонони) | Φ (интегрированная информация) | Да |
| Orch-OR (Пенроуз) | Квантовый коллапс в микротрубочках | Частично |
| Глобальное рабочее пространство | Широковещательная активация | Да |
| Панпсихизм | Сознание фундаментально | Нет |

**Эксперименты**:
- Измерение Φ при анестезии
- Поиск квантовой когерентности в мозге

### 8. Структура Вакуума (35%)

```
Источник: "Пустое" пространство
Трансформер: Квантовые флуктуации
Результат: Эффект Казимира, тёмная энергия
```

**Подтверждено**:
- Эффект Казимира (1997): F/A = -π²ℏc/(240d⁴)
- Лэмбовский сдвиг: виртуальные частицы влияют на атом

**Не понято**:
