# Physics Law Evolution System

Самоэволюционирующая система поиска физических законов.

## Структура

```
evolution/
├── *.vibee           # Спецификации (источник истины)
├── README.md         # Документация
└── DISCOVERY.md      # Описание открытия
```

## Спецификации

- `physics_evolution.vibee` - Главная спецификация системы
- `genome.vibee` - Геном физического закона
- `fitness.vibee` - Фитнес-функция
- `mutator.vibee` - Эволюционные операторы
- `heuristics.vibee` - Эволюционирующие эвристики

## Компиляция

```bash
cd src/vibeec
zig build
./zig-out/bin/vibeec gen ../../experiments/physics/evolution/genome.vibee
```

## Главное открытие

```
ℏ²H₀/(cG·mₑ·mₚ²) ≈ 1/2

Предсказание: H₀ = 70.7 км/с/Мпк
Наблюдение: H₀ ≈ 70 км/с/Мпк
Точность: 96%
```

Это соотношение связывает:
- Квантовую механику (ℏ)
- Гравитацию (G)
- Физику частиц (mₑ, mₚ)
- Космологию (H₀)
