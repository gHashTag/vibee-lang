### Открытие: 1/2 = (φ - 1/φ)/2

**Теорема**: Коэффициент 1/2 выражается через золотое сечение φ.

**Доказательство**:

```
1. Из определения φ: φ² = φ + 1
2. Следовательно: φ - 1/φ = φ - (φ - 1) = 1
3. Поэтому: (φ - 1/φ)/2 = 1/2 ∎
```

**Альтернативная форма**:
```
1/2 = (φ² - 1)/(2φ) = φ/2 - 1/(2φ)
```

### Математическая связь

```
Золотое сечение: φ = (1 + √5)/2 ≈ 1.618

Свойства:
• φ² = φ + 1
• 1/φ = φ - 1
• φ - 1/φ = 1

Асимметрия паттерна создания:
• (φ - 1/φ)/2 = 1/2
```

```

Прямой процесс:   S → T → R  (доля φ)

Асимметрия = (φ - 1/φ)/2 = 1/2

Вселенная расширяется (H₀ > 0) из-за этой асимметрии.
```
