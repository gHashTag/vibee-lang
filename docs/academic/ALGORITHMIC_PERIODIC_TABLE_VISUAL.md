## 🎯 КЛЮЧЕВЫЕ ИНСАЙТЫ ДЛЯ PAS

### 1. Закономерности прорывов

```
ПРАВИЛО 1: Прорывы в умножении следуют паттерну D&C + ALG
           → Ищи алгебраические тождества, уменьшающие число операций

ПРАВИЛО 2: Частотные методы (FDT) применимы везде, где есть свёртка
           → Умножение = свёртка коэффициентов

ПРАВИЛО 3: ML-методы (MLS) эффективны для дискретной оптимизации
           → Формулируй задачу как игру с чётким reward

ПРАВИЛО 4: Предвычисление (PRE) работает при повторяющихся запросах
           → Строй структуры данных для амортизации
```

### 2. Предсказание следующих прорывов

| Область | Текущее состояние | Теор. предел | Вероятный паттерн |
|---------|-------------------|--------------|-------------------|
| Матрицы | O(n^2.38) | O(n²) | MLS + TEN + ALG |
| Графы (APSP) | O(n³) | O(n^2.5)? | MLS + ALG |
| SAT | Exponential | ? | MLS + PRB |
| Convex Hull | O(n log n) | Ω(n log n) | ✓ Оптимально |

### 3. Временные циклы прорывов

```
Среднее время между прорывами по категориям:
- Умножение чисел: ~20 лет (1960→1971→2019)
- Умножение матриц: ~18 лет (1969→1987→2022)  
- Сортировка: ~30 лет (1959→2002→2023)
- Строки: ~7 лет (1970→1975→1987)

→ ML-эра сокращает циклы до 1-5 лет
```
