### 1.1 Связь 3 и φ

$$\boxed{\varphi^2 + \frac{1}{\varphi^2} = 3}$$

**Доказательство:**
```
φ² = φ + 1
1/φ = φ - 1
1/φ² = (φ - 1)² = φ² - 2φ + 1 = φ + 1 - 2φ + 1 = 2 - φ

φ² + 1/φ² = (φ + 1) + (2 - φ) = 3 ∎
```
