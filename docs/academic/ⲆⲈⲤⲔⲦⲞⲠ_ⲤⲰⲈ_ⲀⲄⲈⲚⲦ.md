## Usage

```zig
// Initialize agent
var agent = try DesktopSWEAgent.init(allocator, "desktop-swe-1", llm_config, "/workspaces/vibee-lang");
defer agent.deinit();

// Start agent
_ = try agent.start();

// Process task
const response = try agent.processRequest("implement user authentication");
defer allocator.free(response);

// Self-improve
try agent.improveCycle();

// Commit changes
_ = try agent.commitChanges(&files, "Add authentication system");
```
