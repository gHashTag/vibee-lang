## 1. ORIGINAL KOIDE FORMULA (1982)

### 1.1 Definition

$$Q = \frac{m_e + m_\mu + m_\tau}{(\sqrt{m_e} + \sqrt{m_\mu} + \sqrt{m_\tau})^2}$$

### 1.2 Experimental Values

| Lepton | Mass (MeV) | √Mass |
|--------|------------|-------|
| e | 0.51099895 | 0.71484 |
| μ | 105.6583755 | 10.2787 |
| τ | 1776.86 | 42.1528 |

### 1.3 Calculation

Numerator: 0.511 + 105.658 + 1776.86 = **1883.029**

Denominator: (0.715 + 10.279 + 42.153)² = 53.147² = **2824.60**

Q = 1883.029 / 2824.60 = **0.666661**

### 1.4 Theoretical Value

Q = 2/3 = 0.666666...

### 1.5 Error

|0.666661 - 0.666667| / 0.666667 × 100% = **0.0009%**

## 4. GENERALIZED KOIDE FORMULA

### 4.1 General Form

$$Q_n = \frac{\sum_i m_i}{(\sum_i m_i^{1/n})^n}$$

For n = 2 (original Koide):
$$Q_2 = \frac{\sum m}{(\sum \sqrt{m})^2}$$

### 4.2 Bounds

For any positive masses:
$$\frac{1}{3} \leq Q \leq 1$$

- Q = 1/3: All masses equal
- Q = 1: One mass dominates

### 4.3 Lepton Value

Q = 2/3 is exactly in the middle of the allowed range!
