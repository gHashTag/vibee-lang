# Глава 169: Серый Волк

## Медное царство

> Число 169

### Истина главы

В главе 169 Иван познаёт святой Серый волк.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 169: Серый волк
```
