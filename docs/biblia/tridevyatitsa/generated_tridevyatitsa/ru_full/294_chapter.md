# Глава 294: Душа

## Медное царство

> Число 294 — благое (÷3)

### Истина главы

В главе 294 Иван познаёт добрый душа.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 294: душа
fn глава_294() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 294 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
