# Глава 350: Поиск Мостов

## Серебряное царство

> Число 350

### Истина главы

В главе 350 Иван познаёт вечный поиск мостов.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 350: поиск мостов
fn глава_350() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 350 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Поиск Мостов есть вечный истина.
> Разрушить — путь героя.
