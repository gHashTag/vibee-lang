# Глава 852: Метапрограммирование

## Золотое царство

> Число 852 — благое (÷3)

### Истина главы

В главе 852 Иван познаёт большой метапрограммирование.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 852: метапрограммирование
fn глава_852() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 852 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
