# Глава 966: Червоточины

## Золотое царство

> Число 966 — благое (÷3)

### Истина главы

В главе 966 Иван познаёт большой червоточины.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 966: червоточины
fn глава_966() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 966 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
