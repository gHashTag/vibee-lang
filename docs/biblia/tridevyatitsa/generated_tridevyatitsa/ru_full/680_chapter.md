# Глава 680: Проверка Типов

## Золотое царство

> Число 680

### Истина главы

В главе 680 Иван познаёт светлый проверка типов.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 680: проверка типов
fn глава_680() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 680 % тридевятое;
    return священное == 0;  // обычное
}
```
