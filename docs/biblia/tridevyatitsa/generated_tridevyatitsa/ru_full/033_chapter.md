# Глава 33: Жар-Птица

## Медное царство

> Число 33 — благое (÷3)

### Истина главы

В главе 33 Иван познаёт добрый Жар-птица.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 33: Жар-птица
fn глава_33() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 33 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
