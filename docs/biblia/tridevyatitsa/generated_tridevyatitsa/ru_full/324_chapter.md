# Глава 324: Гамаюн

## Медное царство

> Число 324 — священное (÷27)

### Истина главы

В главе 324 Иван познаёт добрый Гамаюн.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 324: Гамаюн
fn глава_324() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 324 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
