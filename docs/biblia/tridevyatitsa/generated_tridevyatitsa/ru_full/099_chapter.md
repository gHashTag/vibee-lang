# Глава 99: Цветик-Семицветик

## Медное царство

> Число 99 — благословенное (÷9)

### Истина главы

В главе 99 Иван познаёт живой цветик-семицветик.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 99: цветик-семицветик
fn глава_99() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 99 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
