# Глава 576: Avl-Дерево

## Серебряное царство

> Число 576 — благословенное (÷9)

### Истина главы

В главе 576 Иван познаёт светлый AVL-дерево.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 576: AVL-дерево
fn глава_576() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 576 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
