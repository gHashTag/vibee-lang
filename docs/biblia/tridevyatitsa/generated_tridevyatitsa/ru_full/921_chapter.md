# Глава 921: Цифровая Подпись

## Золотое царство

> Число 921 — благое (÷3)

### Истина главы

В главе 921 Иван познаёт вечный цифровая подпись.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 921: цифровая подпись
fn глава_921() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 921 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
