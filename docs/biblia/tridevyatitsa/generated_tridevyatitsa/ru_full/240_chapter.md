# Глава 240: Посох Мудрости

## Медное царство

> Число 240 — благое (÷3)

### Истина главы

В главе 240 Иван познаёт большой посох мудрости.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 240: посох мудрости
fn глава_240() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 240 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Посох Мудрости есть большой истина.
> Познать — путь героя.
> Делай добро.
