# Глава 380: Вещественное Число

## Серебряное царство

> Число 380

### Истина главы

В главе 380 Иван познаёт добрый вещественное число.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 380: вещественное число
fn глава_380() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 380 % тридевятое;
    return священное == 0;  // обычное
}
```
