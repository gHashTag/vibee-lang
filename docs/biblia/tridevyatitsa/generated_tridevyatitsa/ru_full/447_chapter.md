# Глава 447: Граф

## Серебряное царство

> Число 447 — благое (÷3)

### Истина главы

В главе 447 Иван познаёт живой граф.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 447: граф
fn глава_447() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 447 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
