# Глава 256: Мудрость

## Медное царство

> Число 256

### Истина главы

В главе 256 Иван познаёт светлый мудрость.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 256: мудрость
fn глава_256() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 256 % тридевятое;
    return священное == 0;  // обычное
}
```
