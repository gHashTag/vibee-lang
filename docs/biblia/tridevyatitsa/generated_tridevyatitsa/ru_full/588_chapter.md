# Глава 588: Мультимножество

## Серебряное царство

> Число 588 — благое (÷3)

### Истина главы

В главе 588 Иван познаёт живой мультимножество.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 588: мультимножество
fn глава_588() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 588 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
