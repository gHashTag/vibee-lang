# Глава 339: Суффиксный Массив

## Серебряное царство

> Число 339 — благое (÷3)

### Истина главы

В главе 339 Иван познаёт добрый суффиксный массив.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 339: суффиксный массив
fn глава_339() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 339 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Суффиксный Массив есть добрый истина.
> Разрушить — путь героя.
> Делай добро.
