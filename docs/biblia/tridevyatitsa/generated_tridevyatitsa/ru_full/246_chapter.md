# Глава 246: Правда

## Медное царство

> Число 246 — благое (÷3)

### Истина главы

В главе 246 Иван познаёт большой правда.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 246: правда
fn глава_246() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 246 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
