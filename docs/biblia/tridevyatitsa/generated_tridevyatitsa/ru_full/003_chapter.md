# Глава 3: Двойственность

## Медное царство

> Число 3 — благое (÷3)

### Истина главы

В главе 3 Иван познаёт мудрый двойственность.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 3: двойственность
fn глава_3() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 3 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Двойственность есть мудрый истина.
> Разрушить — путь героя.
> Делай добро.
