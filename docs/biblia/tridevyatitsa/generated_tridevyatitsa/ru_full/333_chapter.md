# Глава 333: Правь

## Медное царство

> Число 333 — благословенное (÷9)

### Истина главы

В главе 333 Иван познаёт тёмный Правь.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 333: Правь
fn глава_333() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 333 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

### Мудрость

> Правь есть тёмный истина.
> Разделить — путь героя.
> Делай добро.
