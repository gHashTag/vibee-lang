# Глава 729: Flow Control

## Золотое царство

> Число 729 — священное (÷27)

### Истина главы

В главе 729 Иван познаёт вечный flow control.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 729: flow control
fn глава_729() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 729 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
