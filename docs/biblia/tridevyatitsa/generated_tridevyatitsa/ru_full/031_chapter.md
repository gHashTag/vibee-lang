# Глава 31: Кощей Бессмертный

## Медное царство

> Число 31

### Истина главы

В главе 31 Иван познаёт большой Кощей Бессмертный.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 31: Кощей Бессмертный
fn глава_31() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 31 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Кощей Бессмертный есть большой истина.
> Скрыть — путь героя.
