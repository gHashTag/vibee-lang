# Глава 84: Сапоги-Скороходы

## Медное царство

> Число 84 — благое (÷3)

### Истина главы

В главе 84 Иван познаёт святой сапоги-скороходы.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 84: сапоги-скороходы
fn глава_84() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 84 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
