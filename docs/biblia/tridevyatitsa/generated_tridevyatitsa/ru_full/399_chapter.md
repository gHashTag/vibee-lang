# Глава 399: Promise

## Серебряное царство

> Число 399 — благое (÷3)

### Истина главы

В главе 399 Иван познаёт большой Promise.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 399: Promise
fn глава_399() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 399 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Promise есть большой истина.
> Разрушить — путь героя.
> Делай добро.
