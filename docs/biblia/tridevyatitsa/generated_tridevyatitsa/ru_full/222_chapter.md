# Глава 222: Гусли-Самогуды

## Медное царство

> Число 222 — благое (÷3)

### Истина главы

В главе 222 Иван познаёт малый гусли-самогуды.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: малый конец
