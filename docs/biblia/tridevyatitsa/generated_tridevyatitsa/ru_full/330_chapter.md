# Глава 330: Река Смородина

## Медное царство

> Число 330 — благое (÷3)

### Истина главы

В главе 330 Иван познаёт малый река Смородина.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 330: река Смородина
fn глава_330() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 330 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Река Смородина есть малый истина.
