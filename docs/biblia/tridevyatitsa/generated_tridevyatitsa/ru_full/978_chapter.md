# Глава 978: Нейроморфная Память

## Золотое царство

> Число 978 — благое (÷3)

### Истина главы

В главе 978 Иван познаёт добрый нейроморфная память.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 978: нейроморфная память
fn глава_978() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 978 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
