# Глава 212: Заколдованный Лес

## Медное царство

> Число 212

### Истина главы

В главе 212 Иван познаёт живой заколдованный лес.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 212: заколдованный лес
fn глава_212() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 212 % тридевятое;
    return священное == 0;  // обычное
}
```
