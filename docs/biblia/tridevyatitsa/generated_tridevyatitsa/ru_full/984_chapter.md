# Глава 984: Эмоциональный Процессор

## Золотое царство

> Число 984 — благое (÷3)

### Истина главы

В главе 984 Иван познаёт малый эмоциональный процессор.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 984: эмоциональный процессор
fn глава_984() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 984 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
