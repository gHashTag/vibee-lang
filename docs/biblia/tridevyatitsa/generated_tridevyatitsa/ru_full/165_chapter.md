# Глава 165: Баба-Яга

## Медное царство

> Число 165 — благое (÷3)

### Истина главы

В главе 165 Иван познаёт большой Баба-Яга.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 165: Баба-Яга
fn глава_165() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 165 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Баба-Яга есть большой истина.
> Познать — путь героя.
