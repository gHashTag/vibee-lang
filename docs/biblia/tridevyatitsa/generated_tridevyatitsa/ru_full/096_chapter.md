# Глава 96: Мёртвая Вода

## Медное царство

> Число 96 — благое (÷3)

### Истина главы

В главе 96 Иван познаёт вечный мёртвая вода.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 96: мёртвая вода
fn глава_96() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 96 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Мёртвая Вода есть вечный истина.
> Разделить — путь героя.
> Делай добро.
