# Глава 36: Царевна-Лягушка

## Медное царство

> Число 36 — благословенное (÷9)

### Истина главы

В главе 36 Иван познаёт вечный Царевна-лягушка.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 36: Царевна-лягушка
fn глава_36() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 36 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

> Царевна-Лягушка есть вечный истина.
> Познать — путь героя.
