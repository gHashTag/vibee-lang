## Серебряное царство

> Число 666 — благословенное (÷9)

### Истина главы

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: светлый конец

```vibee
// Глава 666: Either

}
```
