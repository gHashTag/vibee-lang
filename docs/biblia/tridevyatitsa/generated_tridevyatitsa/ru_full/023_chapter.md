# Глава 23: Дух

## Медное царство

> Число 23

### Истина главы

В главе 23 Иван познаёт святой дух.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 23: дух
fn глава_23() {

    let священное = 23 % тридевятое;

}
```
