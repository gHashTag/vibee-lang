# Глава 156: Высота

## Медное царство

> Число 156 — благое (÷3)

### Истина главы

В главе 156 Иван познаёт светлый высота.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 156: высота
fn глава_156() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 156 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Высота есть светлый истина.
