# Глава 51: Берегиня

## Медное царство

> Число 51 — благое (÷3)

### Истина главы

В главе 51 Иван познаёт мудрый Берегиня.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 51: Берегиня
fn глава_51() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 51 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Берегиня есть мудрый истина.
> Разрушить — путь героя.
> Делай добро.
