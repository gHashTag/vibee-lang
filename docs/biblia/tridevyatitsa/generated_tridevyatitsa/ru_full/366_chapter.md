# Глава 366: Команда

## Серебряное царство

> Число 366 — благое (÷3)

### Истина главы

В главе 366 Иван познаёт малый команда.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 366: команда
fn глава_366() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 366 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
