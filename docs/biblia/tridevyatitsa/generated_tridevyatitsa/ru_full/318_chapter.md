# Глава 318: Домовой

## Медное царство

> Число 318 — благое (÷3)

### Истина главы

В главе 318 Иван познаёт большой Домовой.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 318: Домовой
fn глава_318() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 318 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
