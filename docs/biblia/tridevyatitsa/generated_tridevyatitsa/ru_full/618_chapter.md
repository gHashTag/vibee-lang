# Глава 618: Поиск Цикла

## Серебряное царство

> Число 618 — благое (÷3)

### Истина главы

В главе 618 Иван познаёт светлый поиск цикла.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 618: поиск цикла
fn глава_618() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 618 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Поиск Цикла есть светлый истина.
> Разрушить — путь героя.
