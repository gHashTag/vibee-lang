# Глава 111: Правда

## Медное царство

> Число 111 — благое (÷3)

### Истина главы

В главе 111 Иван познаёт мудрый правда.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: мудрый конец

```vibee
// Глава 111: правда
fn глава_111() {

}
```
