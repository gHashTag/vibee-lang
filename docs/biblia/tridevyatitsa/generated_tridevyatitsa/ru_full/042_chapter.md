# Глава 42: Алёша Попович

## Медное царство

> Число 42 — благое (÷3)

### Истина главы

В главе 42 Иван познаёт малый Алёша Попович.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 42: Алёша Попович
fn глава_42() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 42 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
