# Глава 204: Тёмный Терем

## Медное царство

> Число 204 — благое (÷3)

### Истина главы

В главе 204 Иван познаёт живой тёмный терем.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 204: тёмный терем
fn глава_204() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 204 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
