# Глава 411: Сортировка Подсчётом

## Серебряное царство

> Число 411 — благое (÷3)

### Истина главы

В главе 411 Иван познаёт мудрый сортировка подсчётом.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 411: сортировка подсчётом
fn глава_411() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 411 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
