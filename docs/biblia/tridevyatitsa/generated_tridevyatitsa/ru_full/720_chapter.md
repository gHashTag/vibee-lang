# Глава 720: Корутины

## Золотое царство

> Число 720 — благословенное (÷9)

### Истина главы

В главе 720 Иван познаёт добрый корутины.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 720: корутины
fn глава_720() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 720 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

> Корутины есть добрый истина.
> Познать — путь героя.
