# Глава 594: Rope

## Серебряное царство

> Число 594 — священное (÷27)

### Истина главы

В главе 594 Иван познаёт добрый rope.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 594: rope
fn глава_594() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 594 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
