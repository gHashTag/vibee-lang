# Глава 630: Компоновщик

## Серебряное царство

> Число 630 — благословенное (÷9)

### Истина главы

В главе 630 Иван познаёт светлый компоновщик.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 630: компоновщик
fn глава_630() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 630 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

### Мудрость

> Компоновщик есть светлый истина.
> Разрушить — путь героя.
> Делай добро.
