### Истина главы

В главе 130 Иван познаёт тёмный тьма.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 130: тьма
fn глава_130() {

    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 130 % тридевятое;

}
```
