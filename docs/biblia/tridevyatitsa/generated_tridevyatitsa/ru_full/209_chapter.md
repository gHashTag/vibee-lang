# Глава 209: Подводное Царство

## Медное царство

> Число 209

### Истина главы

В главе 209 Иван познаёт тёмный подводное царство.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 209: подводное царство
fn глава_209() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 209 % тридевятое;
    return священное == 0;  // обычное
}
```
