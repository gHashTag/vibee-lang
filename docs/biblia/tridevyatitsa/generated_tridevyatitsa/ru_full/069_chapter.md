# Глава 69: Тёмный Терем

## Медное царство

> Число 69 — благое (÷3)

### Истина главы

В главе 69 Иван познаёт большой тёмный терем.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 69: тёмный терем
fn глава_69() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 69 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Тёмный Терем есть большой истина.
> Создать — путь героя.
