# Глава 296: Царство

## Медное царство

> Число 296

### Истина главы

В главе 296 Иван познаёт малый царство.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 296: царство
fn глава_296() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 296 % тридевятое;
    return священное == 0;  // обычное
}
```
