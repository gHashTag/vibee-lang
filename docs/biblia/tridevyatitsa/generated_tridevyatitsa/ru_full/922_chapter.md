# Глава 922: Сертификат

## Золотое царство

> Число 922

### Истина главы

В главе 922 Иван познаёт большой сертификат.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 922: сертификат

}
```
