# Глава 990: Вселенский Стандарт

## Золотое царство

> Число 990 — благословенное (÷9)

### Истина главы

В главе 990 Иван познаёт тёмный вселенский стандарт.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 990: вселенский стандарт
fn глава_990() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 990 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
