# Глава 214: Кисельные Берега

## Медное царство

> Число 214

### Истина главы

В главе 214 Иван познаёт тёмный кисельные берега.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 214: кисельные берега

}
```
