# Глава 963: Терраформирование

## Золотое царство

> Число 963 — благословенное (÷9)

### Истина главы

В главе 963 Иван познаёт тёмный терраформирование.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 963: терраформирование
fn глава_963() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 963 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
