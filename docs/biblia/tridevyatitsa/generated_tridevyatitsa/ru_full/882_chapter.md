# Глава 882: Корутины

## Золотое царство

> Число 882 — благословенное (÷9)

### Истина главы

В главе 882 Иван познаёт большой корутины.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 882: корутины
fn глава_882() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 882 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
