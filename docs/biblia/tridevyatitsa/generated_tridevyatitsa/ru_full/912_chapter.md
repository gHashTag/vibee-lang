# Глава 912: Dht

## Золотое царство

> Число 912 — благое (÷3)

### Истина главы

В главе 912 Иван познаёт тёмный DHT.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 912: DHT
fn глава_912() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 912 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
