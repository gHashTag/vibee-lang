# Глава 282: Свет

## Медное царство

> Число 282 — благое (÷3)

### Истина главы

В главе 282 Иван познаёт большой свет.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 282: свет
fn глава_282() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 282 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Свет есть большой истина.
> Скрыть — путь героя.
