# Глава 19: Слово

## Медное царство

> Число 19

### Истина главы

В главе 19 Иван познаёт большой слово.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 19: слово
fn глава_19() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 19 % тридевятое;
    return священное == 0;  // обычное
}
```
