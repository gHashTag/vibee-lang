# Глава 480: R-Дерево

## Серебряное царство

> Число 480 — благое (÷3)

### Истина главы

В главе 480 Иван познаёт живой R-дерево.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 480: R-дерево
fn глава_480() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 480 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> R-Дерево есть живой истина.
> Соединить — путь героя.
> Делай добро.
