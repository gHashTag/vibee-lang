# Глава 486: Поиск Точек Сочленения

## Серебряное царство

> Число 486 — священное (÷27)

### Истина главы

В главе 486 Иван познаёт добрый поиск точек сочленения.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 486: поиск точек сочленения
fn глава_486() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 486 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

### Мудрость

> Поиск Точек Сочленения есть добрый истина.
> Разрушить — путь героя.
> Делай добро.
