# Глава 414: Сортировка Выбором

## Серебряное царство

> Число 414 — благословенное (÷9)

### Истина главы

В главе 414 Иван познаёт живой сортировка выбором.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 414: сортировка выбором
fn глава_414() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 414 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
