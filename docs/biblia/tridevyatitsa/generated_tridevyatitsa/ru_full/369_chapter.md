# Глава 369: Хранитель

## Серебряное царство

> Число 369 — благословенное (÷9)

### Истина главы

В главе 369 Иван познаёт вечный хранитель.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 369: хранитель
fn глава_369() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 369 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

> Хранитель есть вечный истина.
