# Глава 354: Абстрактная Фабрика

## Серебряное царство

> Число 354 — благое (÷3)

### Истина главы

### Три аспекта

3. **Третий аспект**: мудрый конец

// Глава 354: абстрактная фабрика
fn глава_354() {

    let тридевятое = тройка * тройка * тройка;  // 27
