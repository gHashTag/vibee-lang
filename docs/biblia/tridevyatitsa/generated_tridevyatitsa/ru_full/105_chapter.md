# Глава 105: Посох Мудрости

## Медное царство

> Число 105 — благое (÷3)

### Истина главы

В главе 105 Иван познаёт вечный посох мудрости.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 105: посох мудрости
fn глава_105() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 105 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
