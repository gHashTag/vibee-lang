# Глава 927: Abac

## Золотое царство

> Число 927 — благословенное (÷9)

### Истина главы

В главе 927 Иван познаёт добрый ABAC.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 927: ABAC
fn глава_927() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 927 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
