# Глава 270: Бессмертие

## Медное царство

> Число 270 — священное (÷27)

### Истина главы

В главе 270 Иван познаёт добрый бессмертие.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 270: бессмертие
fn глава_270() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 270 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

### Мудрость

> Бессмертие есть добрый истина.
> Разделить — путь героя.
> Делай добро.
