# Глава 15: Порядок

## Медное царство

> Число 15 — благое (÷3)

### Истина главы

В главе 15 Иван познаёт мудрый порядок.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 15: порядок
fn глава_15() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 15 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Порядок есть мудрый истина.
> Познать — путь героя.
