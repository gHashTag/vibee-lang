# Глава 303: Жар-Птица

## Медное царство

> Число 303 — благое (÷3)

### Истина главы

В главе 303 Иван познаёт большой Жар-птица.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 303: Жар-птица
fn глава_303() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 303 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
