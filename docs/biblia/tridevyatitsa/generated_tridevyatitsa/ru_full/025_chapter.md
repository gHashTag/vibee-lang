# Глава 25: Вечность

## Медное царство

> Число 25

### Истина главы

В главе 25 Иван познаёт живой вечность.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 25: вечность
fn глава_25() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 25 % тридевятое;
    return священное == 0;  // обычное
}
```
