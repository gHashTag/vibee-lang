# Глава 810: Дополненная Реальность

## Золотое царство

> Число 810 — священное (÷27)

### Истина главы

В главе 810 Иван познаёт светлый дополненная реальность.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 810: дополненная реальность
fn глава_810() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 810 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
