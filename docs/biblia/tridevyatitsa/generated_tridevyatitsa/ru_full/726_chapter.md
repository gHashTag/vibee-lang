# Глава 726: Event Loop

## Золотое царство

> Число 726 — благое (÷3)

### Истина главы

В главе 726 Иван познаёт святой event loop.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 726: event loop
fn глава_726() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 726 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Event Loop есть святой истина.
> Разделить — путь героя.
