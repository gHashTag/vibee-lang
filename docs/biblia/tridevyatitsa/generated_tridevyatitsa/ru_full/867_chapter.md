# Глава 867: Стековая Машина

## Золотое царство

> Число 867 — благое (÷3)

### Истина главы

В главе 867 Иван познаёт добрый стековая машина.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 867: стековая машина
fn глава_867() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 867 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
