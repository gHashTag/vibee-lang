# Глава 255: Месть

## Медное царство

> Число 255 — благое (÷3)

### Истина главы

В главе 255 Иван познаёт большой месть.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 255: месть
fn глава_255() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 255 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Месть есть большой истина.
> Разделить — путь героя.
