# Глава 207: Хрустальный Дворец

## Медное царство

> Число 207 — благословенное (÷9)

### Истина главы

В главе 207 Иван познаёт тёмный хрустальный дворец.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 207: хрустальный дворец
fn глава_207() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 207 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
