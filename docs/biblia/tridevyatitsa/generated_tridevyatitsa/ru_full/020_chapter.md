# Глава 20: Твердь

## Медное царство

> Число 20

### Истина главы

В главе 20 Иван познаёт добрый твердь.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 20: твердь
fn глава_20() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 20 % тридевятое;
    return священное == 0;  // обычное
}
```
