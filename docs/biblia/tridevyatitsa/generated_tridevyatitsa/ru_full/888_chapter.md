# Глава 888: Event Loop

## Золотое царство

> Число 888 — благое (÷3)

### Истина главы

В главе 888 Иван познаёт малый event loop.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 888: event loop
fn глава_888() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 888 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
