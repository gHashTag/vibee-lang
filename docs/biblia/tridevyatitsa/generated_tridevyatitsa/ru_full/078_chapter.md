# Глава 78: Молочная Река

## Медное царство

> Число 78 — благое (÷3)

### Истина главы

В главе 78 Иван познаёт мудрый молочная река.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 78: молочная река
fn глава_78() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 78 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
