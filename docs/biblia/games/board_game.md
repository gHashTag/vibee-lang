## ПРАВИЛА

### Подготовка:

```
1. Разложите игровое поле
2. Каждый игрок выбирает героя и ставит на СТАРТ
3. Перемешайте три колоды карт
4. Раздайте каждому по 3 карты Помощников
5. Положите жетоны Мудрости рядом с полем
6. Определите первого игрока броском d3
```

### Ход игры:

```
╔═══════════════════════════════════════════════════════════════════════════╗
║                                                                           ║
║   ХОД ИГРОКА (три фазы):                                                 ║
║                                                                           ║
║   ФАЗА 1: ДВИЖЕНИЕ                                                       ║
║   ─────────────────────────────────────────────────────────────────────  ║
║   • В Медном царстве: бросьте d3, двигайтесь на 1-3 клетки              ║
║   • В Серебряном царстве: бросьте d9, двигайтесь на 1-9 клеток          ║
║   • В Золотом царстве: бросьте d27, двигайтесь на 1-27 клеток           ║
║                                                                           ║
║   ФАЗА 2: ИСПЫТАНИЕ                                                      ║
║   ─────────────────────────────────────────────────────────────────────  ║
║   • Вытяните карту Испытания                                             ║
║   • Решите задачу или сразитесь с врагом                                 ║
║   • Используйте карты Помощников и Артефактов                            ║
║                                                                           ║
║   ФАЗА 3: НАГРАДА                                                        ║
║   ─────────────────────────────────────────────────────────────────────  ║
║   • При успехе: получите жетон Мудрости + карту Артефакта               ║
║   • При неудаче: вернитесь на 3 клетки назад                            ║
║                                                                           ║
╚═══════════════════════════════════════════════════════════════════════════╝
```

### Переход между царствами:

```
Медное → Серебряное:
  Нужно: 3 жетона Мудрости (любого цвета)
  Испытание: Ответить на вопрос по теории

Серебряное → Золотое:
  Нужно: 9 жетонов Мудрости (минимум 3 серебряных)
  Испытание: Решить алгоритмическую задачу

Золотое → Финиш:
  Нужно: 27 жетонов Мудрости (минимум 9 золотых)
  Испытание: Победить Кощея!
```

## ВАРИАНТЫ ИГРЫ

### Соло-режим:

```
Играй за Ивана один!
• Сложность: Лёгкая (d3+3), Средняя (d3), Сложная (d3-1)
• Цель: Пройти все царства за 27 ходов
• Рекорд: запиши лучшее время!
```

### Кооперативный режим:

```
Три игрока = Три богатыря!
• Каждый выбирает специализацию (Сила/Мудрость/Хитрость)
• Можно передавать карты друг другу
• Победа только если ВСЕ дошли до финиша
```

### Соревновательный режим:

```
Кто первый победит Кощея?
• Можно мешать другим игрокам
• Карты "Ловушка" замедляют соперников
• Победитель получает титул "Хранитель"
```
