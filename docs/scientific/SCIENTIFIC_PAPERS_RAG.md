## Core RAG Papers

### 1. RAG: Retrieval-Augmented Generation
**Lewis et al. 2020** (Facebook AI)
- Paper: "Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks"
- Venue: NeurIPS 2020
- Key contribution: First RAG architecture combining retrieval with generation
- Citation: 3000+

### 2. REALM: Retrieval-Augmented Language Model Pre-Training
**Guu et al. 2020** (Google)
- Paper: "REALM: Retrieval-Augmented Language Model Pre-Training"
- Venue: ICML 2020
- Key contribution: Pre-training with retrieval

### 3. FiD: Fusion-in-Decoder
**Izacard & Grave 2021** (Facebook AI)
- Paper: "Leveraging Passage Retrieval with Generative Models"
- Venue: EACL 2021
- Key contribution: Efficient multi-passage fusion

## Embedding Papers

### 4. Sentence-BERT
**Reimers & Gurevych 2019** (UKP Lab)
- Paper: "Sentence-BERT: Sentence Embeddings using Siamese BERT-Networks"
- Venue: EMNLP 2019
- Key contribution: Efficient sentence embeddings

### 5. MiniLM
**Wang et al. 2020** (Microsoft)
- Paper: "MiniLM: Deep Self-Attention Distillation"
- Venue: NeurIPS 2020
- Key contribution: Compact transformer distillation

### 6. SimCSE
**Gao et al. 2021** (Princeton)
- Paper: "SimCSE: Simple Contrastive Learning of Sentence Embeddings"
- Venue: EMNLP 2021
- Key contribution: Contrastive learning for embeddings

## Search Papers

### 7. BM25

- Paper: "The Probabilistic Relevance Framework: BM25 and Beyond"

**Karpukhin et al. 2020** (Facebook AI)
- Paper: "Dense Passage Retrieval for Open-Domain QA"
- Venue: EMNLP 2020
- Key contribution: Dense retrieval for QA

### 9. Hybrid Search
**Ma et al. 2021**
- Paper: "A Replication Study of Dense Passage Retriever"
- Key contribution: BM25 + Dense combination
