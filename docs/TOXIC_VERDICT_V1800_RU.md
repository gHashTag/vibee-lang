### 6. State Space Models (v1431-v1434)

```
СЛОЖНОСТЬ ВНИМАНИЯ:
  Transformer: O(N²) - квадратичная
  Mamba/S4:    O(N)  - линейная

СКОРОСТЬ:
  Transformer @ 1M tokens: 100 сек
  Mamba @ 1M tokens:       20 сек (5x быстрее!)

ПАМЯТЬ:
  Transformer: O(N) на токен
  Mamba:       O(1) на токен
```

**Вердикт**: SSM заменяют Transformer для длинных контекстов!
