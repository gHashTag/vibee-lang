## Glassmorphism Design System

### Core Principles
```
1. Backdrop Blur: 20-40px blur radius
2. Transparency: 60-80% opacity
3. Soft Shadows: Subtle, diffused shadows
4. Gradient Overlays: Subtle color gradients
5. Border Radius: 12-24px rounded corners
6. Animations: 0.2-0.4s smooth transitions
```

### Color Palette
```
Background: rgba(255, 255, 255, 0.7)
Surface: rgba(255, 255, 255, 0.5)
Border: rgba(255, 255, 255, 0.3)
Shadow: rgba(0, 0, 0, 0.1)
Text: #1d1d1f (Apple Dark)
Accent: #007AFF (Apple Blue)
```

### Typography
```
Font Family: SF Pro Display, SF Pro Text
Base Size: 17px
Line Height: 1.47
Letter Spacing: -0.022em
```
