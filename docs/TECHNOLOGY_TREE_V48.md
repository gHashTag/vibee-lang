## Detailed Learning Path

### Level 1: Foundations (2-4 weeks)

| Topic | Resources | Practice |
|-------|-----------|----------|
| **Zig Language** | ziglearn.org, Zig docs | Implement basic data structures |
| **Memory Model** | "What Every Programmer Should Know About Memory" | Write allocator |
| **POSIX API** | Stevens "APUE" | File I/O, signals |
| **Terminal I/O** | ANSI escape codes | Build simple TUI |

### Level 2: Data Structures (2-3 weeks)

| Topic | Paper/Book | Implementation |
|-------|------------|----------------|
| **Rope** | Boehm et al. 1995 | Text buffer for editor |
| **Gap Buffer** | Emacs implementation | Simple text editing |
| **B-Tree** | Bayer & McCreight 1972 | Index structure |
| **Hash Table** | Knuth TAOCP Vol 3 | Symbol tables |

### Level 3: Core Algorithms (4-6 weeks)

| Topic | Paper | Complexity | VIBEE Module |
|-------|-------|------------|--------------|
| **LCS/Diff** | Myers 1986 | O(ND) | `pro_code_editor_v48.zig` |
| **String Matching** | KMP 1977, Aho-Corasick 1975 | O(n+m) | Syntax highlighter |
| **Parsing** | Dragon Book | O(n) | `parser_v3.zig` |
| **Type Inference** | Hindley-Milner | O(n) | `type_checker.zig` |

### Level 4: Advanced Topics (6-8 weeks)

| Topic | Paper | Application |
|-------|-------|-------------|
| **SIMD Optimization** | Intel Intrinsics Guide | Fast string compare |
| **ML-Guided Search** | AlphaDev 2023 | Code optimization |
| **JIT Compilation** | Aycock 2003 | Runtime codegen |
| **Incremental Parsing** | Tree-sitter | Real-time syntax |
