1. **VIBEE_FORMAL_SPECIFICATION.md** - Type system, operational semantics
2. **VIBEE_LANGUAGE_STANDARD.md** - Complete language reference  
3. **VIBEE_THEOREMS_AND_PROOFS.md** - Soundness proofs
