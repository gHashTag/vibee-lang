# VIBEE Browser Features

## φ² + 1/φ² = 3 | PHOENIX = 999

## Overview

VIBEE Browser is a Chromium-based browser with integrated AI and development tools, built on the Thorium performance foundation.

## Core Features

### 1. φ-Layout UI System

Golden ratio-based interface layout for optimal visual harmony.

```
┌─────────────────────────────────────────────────────────────┐
│ Navigation Bar (φ-proportioned)                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                                                             │
│                    Main Content                             │
│                    (61.8% width)                            │
│                                                             │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│ Status Bar                                                  │
└─────────────────────────────────────────────────────────────┘
```

**Key proportions:**
- Main content: 61.8% (φ)
- Sidebar: 38.2% (1/φ)
- Tab height: 34px (φ-derived)
- Toolbar height: 55px (φ-derived)

### 2. AI Sidebar

Integrated AI assistant accessible via `Ctrl+Shift+A`.

**Capabilities:**
- Page summarization
- Code explanation
- Translation
- Research assistance
- Content generation

**Providers:**
- OpenAI GPT-4
- Anthropic Claude
- Local LLMs (Ollama)

### 3. Vibecoding Panel

Developer tools panel for VIBEE language development.

**Access:** `Ctrl+Shift+V` or DevTools → Vibecoding tab

**Features:**
- `.vibee` file syntax highlighting
- Live specification preview
- Code generation (Zig, Python, Rust)
- Test runner integration
- PAS analysis tools

### 4. Performance (Thorium Base)

Inherited from Thorium:
- AVX/AVX2 optimizations
- Compiler optimizations (LTO, PGO)
- Memory efficiency improvements
- Faster JavaScript execution

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+A` | Toggle AI Sidebar |
| `Ctrl+Shift+V` | Toggle Vibecoding Panel |
| `Ctrl+Shift+P` | Toggle φ-Layout |
| `F12` | DevTools |
| `Ctrl+L` | Focus address bar |

## Configuration

### AI Settings

```
Settings → VIBEE → AI Sidebar
```

- API Key configuration
- Model selection
- Context length
- Temperature

### φ-Layout Settings

```
Settings → VIBEE → φ-Layout
```

- Enable/disable golden ratio
- Custom proportions
- Animation speed

### Vibecoding Settings

```
Settings → VIBEE → Vibecoding
```

- Default language target
- Auto-generate on save
- Test runner path

## Privacy

- AI features are opt-in
- No data collection beyond standard Chromium
- Local LLM support for offline use
- All VIBEE features can be disabled
