# Глава 348: Поиск Цикла

## Серебряное царство

> Число 348 — благое (÷3)

### Истина главы

В главе 348 Иван познаёт вечный поиск цикла.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 348: поиск цикла
fn глава_348() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 348 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
