# Глава 168: Жар-Птица

## Медное царство

> Число 168 — благое (÷3)

### Истина главы

В главе 168 Иван познаёт малый Жар-птица.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 168: Жар-птица
fn глава_168() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 168 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Жар-Птица есть малый истина.
> Разделить — путь героя.
