# Глава 183: Домовой

## Медное царство

> Число 183 — благое (÷3)

### Истина главы

В главе 183 Иван познаёт малый Домовой.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 183: Домовой

}
```
