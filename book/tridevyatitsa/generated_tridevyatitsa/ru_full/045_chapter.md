# Глава 45: Морозко

## Медное царство

> Число 45 — благословенное (÷9)

### Истина главы

В главе 45 Иван познаёт святой Морозко.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 45: Морозко
fn глава_45() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 45 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
