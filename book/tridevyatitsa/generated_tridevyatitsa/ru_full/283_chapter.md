# Глава 283: Мудрость

## Медное царство

> Число 283

### Истина главы

В главе 283 Иван познаёт большой мудрость.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 283: мудрость
fn глава_283() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 283 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Мудрость есть большой истина.
> Разрушить — путь героя.
> Делай добро.
