# Глава 702: Отладка

## Золотое царство

> Число 702 — священное (÷27)

### Истина главы

В главе 702 Иван познаёт малый отладка.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 702: отладка
fn глава_702() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 702 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

### Мудрость

> Отладка есть малый истина.
> Разрушить — путь героя.
> Делай добро.
