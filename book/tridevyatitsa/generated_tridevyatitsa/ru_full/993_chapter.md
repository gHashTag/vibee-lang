# Глава 993: Пространственный Туннель

## Золотое царство

> Число 993 — благое (÷3)

### Истина главы

В главе 993 Иван познаёт большой пространственный туннель.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 993: пространственный туннель
fn глава_993() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 993 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
