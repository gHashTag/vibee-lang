# Глава 417: Гномья Сортировка

## Серебряное царство

> Число 417 — благое (÷3)

### Истина главы

В главе 417 Иван познаёт малый гномья сортировка.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 417: гномья сортировка
fn глава_417() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 417 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
