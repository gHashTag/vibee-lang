# Глава 126: Жадность

## Медное царство

> Число 126 — благословенное (÷9)

### Истина главы

В главе 126 Иван познаёт большой жадность.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 126: жадность
fn глава_126() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 126 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
