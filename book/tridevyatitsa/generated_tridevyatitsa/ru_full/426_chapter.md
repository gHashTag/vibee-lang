# Глава 426: Чётно-Нечётная Сортировка

## Серебряное царство

> Число 426 — благое (÷3)

### Истина главы

В главе 426 Иван познаёт добрый чётно-нечётная сортировка.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 426: чётно-нечётная сортировка
fn глава_426() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 426 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
