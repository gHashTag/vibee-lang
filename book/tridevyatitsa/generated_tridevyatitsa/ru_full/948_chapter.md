# Глава 948: Оптические Компьютеры

## Золотое царство

> Число 948 — благое (÷3)

### Истина главы

В главе 948 Иван познаёт тёмный оптические компьютеры.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 948: оптические компьютеры
fn глава_948() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 948 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
