# Глава 987: Коллективное Сознание

## Золотое царство

> Число 987 — благое (÷3)

### Истина главы

В главе 987 Иван познаёт тёмный коллективное сознание.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 987: коллективное сознание
fn глава_987() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 987 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
