# Глава 516: Комплексное Число

## Серебряное царство

> Число 516 — благое (÷3)

### Истина главы

В главе 516 Иван познаёт мудрый комплексное число.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 516: комплексное число
fn глава_516() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 516 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
