# Глава 825: Коллективное Сознание

## Золотое царство

> Число 825 — благое (÷3)

### Истина главы

В главе 825 Иван познаёт святой коллективное сознание.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 825: коллективное сознание
fn глава_825() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 825 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
