# Глава 137: Единство

## Медное царство

> Число 137

### Истина главы

В главе 137 Иван познаёт малый единство.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 137: единство
fn глава_137() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 137 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Единство есть малый истина.
> Разрушить — путь героя.
