# Глава 297: Полнота

## Медное царство

> Число 297 — священное (÷27)

### Истина главы

В главе 297 Иван познаёт тёмный полнота.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 297: полнота
fn глава_297() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 297 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

### Мудрость

> Полнота есть тёмный истина.
> Разделить — путь героя.
> Делай добро.
