# Глава 636: Команда

## Серебряное царство

> Число 636 — благое (÷3)

### Истина главы

В главе 636 Иван познаёт мудрый команда.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 636: команда
fn глава_636() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 636 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
