# Глава 153: Речь

## Медное царство

> Число 153 — благословенное (÷9)

### Истина главы

В главе 153 Иван познаёт светлый речь.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 153: речь
fn глава_153() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 153 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

### Мудрость

> Речь есть светлый истина.
> Создать — путь героя.
