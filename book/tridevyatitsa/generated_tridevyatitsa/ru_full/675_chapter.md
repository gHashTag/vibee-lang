# Глава 675: Мгновенная Связь

## Золотое царство

> Число 675 — священное (÷27)

### Истина главы

В главе 675 Иван познаёт мудрый мгновенная связь.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 675: мгновенная связь
fn глава_675() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 675 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
