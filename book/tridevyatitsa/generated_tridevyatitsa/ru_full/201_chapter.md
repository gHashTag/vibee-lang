# Глава 201: Синее Море

## Медное царство

> Число 201 — благое (÷3)

### Истина главы

В главе 201 Иван познаёт малый синее море.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 201: синее море
fn глава_201() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 201 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Синее Море есть малый истина.
> Разделить — путь героя.
