# Глава 900: Rpc

## Золотое царство

> Число 900 — благословенное (÷9)

### Истина главы

В главе 900 Иван познаёт мудрый RPC.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 900: RPC
fn глава_900() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 900 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
