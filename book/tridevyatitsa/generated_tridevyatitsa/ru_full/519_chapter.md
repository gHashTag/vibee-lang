# Глава 519: Символ

## Серебряное царство

> Число 519 — благое (÷3)

### Истина главы

В главе 519 Иван познаёт светлый символ.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 519: символ
fn глава_519() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 519 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
