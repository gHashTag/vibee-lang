# Глава 430: Сортировка Спагетти

## Серебряное царство

> Число 430

### Истина главы

В главе 430 Иван познаёт святой сортировка спагетти.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 430: сортировка спагетти
fn глава_430() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 430 % тридевятое;
    return священное == 0;  // обычное
}
```
