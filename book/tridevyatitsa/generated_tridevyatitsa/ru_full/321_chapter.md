# Глава 321: Берегиня

## Медное царство

> Число 321 — благое (÷3)

### Истина главы

В главе 321 Иван познаёт святой Берегиня.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 321: Берегиня
fn глава_321() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 321 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
