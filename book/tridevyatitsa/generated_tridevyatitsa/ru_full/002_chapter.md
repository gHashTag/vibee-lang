# Глава 2: Единство

## Медное царство

> Число 2

### Истина главы

В главе 2 Иван познаёт большой единство.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 2: единство
fn глава_2() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 2 % тридевятое;
    return священное == 0;  // обычное
}
```

> Единство есть большой истина.
