# Глава 822: Эмоциональный Процессор

## Золотое царство

> Число 822 — благое (÷3)

### Истина главы

В главе 822 Иван познаёт малый эмоциональный процессор.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 822: эмоциональный процессор
fn глава_822() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 822 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
