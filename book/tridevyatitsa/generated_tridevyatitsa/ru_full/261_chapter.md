# Глава 261: Жадность

## Медное царство

> Число 261 — благословенное (÷9)

### Истина главы

В главе 261 Иван познаёт живой жадность.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 261: жадность
fn глава_261() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 261 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
