# Глава 6: Жизнь

## Медное царство

> Число 6 — благое (÷3)

### Истина главы

В главе 6 Иван познаёт малый жизнь.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 6: жизнь
fn глава_6() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 6 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Жизнь есть малый истина.
