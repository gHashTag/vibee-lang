# Глава 186: Берегиня

## Медное царство

> Число 186 — благое (÷3)

### Истина главы

В главе 186 Иван познаёт вечный Берегиня.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 186: Берегиня
fn глава_186() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 186 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
