# Глава 933: Zero Trust

## Золотое царство

> Число 933 — благое (÷3)

### Истина главы

В главе 933 Иван познаёт тёмный zero trust.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 933: zero trust
fn глава_933() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 933 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
