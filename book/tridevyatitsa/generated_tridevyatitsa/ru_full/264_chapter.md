# Глава 264: Свет

## Медное царство

> Число 264 — благое (÷3)

### Истина главы

В главе 264 Иван познаёт святой свет.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 264: свет
fn глава_264() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 264 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
