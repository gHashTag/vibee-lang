# Глава 513: Чистая Архитектура

## Серебряное царство

> Число 513 — священное (÷27)

### Истина главы

В главе 513 Иван познаёт большой чистая архитектура.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 513: чистая архитектура
fn глава_513() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 513 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
