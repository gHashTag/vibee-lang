# Глава 567: Квантовая Сортировка

## Серебряное царство

> Число 567 — священное (÷27)

### Истина главы

В главе 567 Иван познаёт добрый квантовая сортировка.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 567: квантовая сортировка
fn глава_567() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 567 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

> Квантовая Сортировка есть добрый истина.
> Разделить — путь героя.
