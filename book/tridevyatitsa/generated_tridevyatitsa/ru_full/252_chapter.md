# Глава 252: Дружба

## Медное царство

> Число 252 — благословенное (÷9)

### Истина главы

В главе 252 Иван познаёт большой дружба.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 252: дружба
fn глава_252() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 252 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

### Мудрость

> Дружба есть большой истина.
> Соединить — путь героя.
> Делай добро.
