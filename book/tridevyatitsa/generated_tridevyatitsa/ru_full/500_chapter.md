# Глава 500: Цепочка Обязанностей

## Серебряное царство

> Число 500

### Истина главы

В главе 500 Иван познаёт светлый цепочка обязанностей.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 500: цепочка обязанностей
fn глава_500() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 500 % тридевятое;
    return священное == 0;  // обычное
}
```
