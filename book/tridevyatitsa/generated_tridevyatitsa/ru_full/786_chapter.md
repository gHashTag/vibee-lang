# Глава 786: Оптические Компьютеры

## Золотое царство

> Число 786 — благое (÷3)

### Истина главы

В главе 786 Иван познаёт вечный оптические компьютеры.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 786: оптические компьютеры
fn глава_786() {

    let тридевятое = тройка * тройка * тройка;  // 27

}
```
