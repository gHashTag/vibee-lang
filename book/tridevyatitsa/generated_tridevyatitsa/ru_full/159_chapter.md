# Глава 159: Душа

## Медное царство

> Число 159 — благое (÷3)

### Истина главы

В главе 159 Иван познаёт живой душа.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 159: душа
fn глава_159() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 159 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Душа есть живой истина.
> Разрушить — путь героя.
> Делай добро.
