# Глава 837: Мгновенная Связь

## Золотое царство

> Число 837 — священное (÷27)

### Истина главы

В главе 837 Иван познаёт тёмный мгновенная связь.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 837: мгновенная связь
fn глава_837() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 837 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

> Соединить — путь героя.
