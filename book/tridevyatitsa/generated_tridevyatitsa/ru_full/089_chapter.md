# Глава 89: Клубок Путеводный

## Медное царство

> Число 89

### Истина главы

В главе 89 Иван познаёт малый клубок путеводный.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 89: клубок путеводный
fn глава_89() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 89 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Клубок Путеводный есть малый истина.
> Соединить — путь героя.
