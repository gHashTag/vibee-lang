# Глава 384: Символ

## Серебряное царство

> Число 384 — благое (÷3)

### Истина главы

В главе 384 Иван познаёт живой символ.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 384: символ
fn глава_384() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 384 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
