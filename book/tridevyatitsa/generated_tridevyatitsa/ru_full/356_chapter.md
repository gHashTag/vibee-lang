# Глава 356: Прототип

## Серебряное царство

> Число 356

### Истина главы

В главе 356 Иван познаёт большой прототип.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 356: прототип

}
```
