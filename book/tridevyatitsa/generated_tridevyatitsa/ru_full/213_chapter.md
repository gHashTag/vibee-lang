# Глава 213: Молочная Река

## Медное царство

> Число 213 — благое (÷3)

### Истина главы

В главе 213 Иван познаёт добрый молочная река.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 213: молочная река
fn глава_213() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 213 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
