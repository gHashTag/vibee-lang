# Глава 73: Подземное Царство

## Медное царство

> Число 73

### Истина главы

В главе 73 Иван познаёт святой подземное царство.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 73: подземное царство
fn глава_73() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 73 % тридевятое;
    return священное == 0;  // обычное
}
```
