# Глава 21: Высота

## Медное царство

> Число 21 — благое (÷3)

### Истина главы

В главе 21 Иван познаёт добрый высота.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 21: высота
fn глава_21() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 21 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Высота есть добрый истина.
> Найти — путь героя.
> Делай добро.
