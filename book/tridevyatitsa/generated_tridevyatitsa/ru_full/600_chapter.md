# Глава 600: A*

## Серебряное царство

> Число 600 — благое (÷3)

### Истина главы

В главе 600 Иван познаёт святой A*.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 600: A*
fn глава_600() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 600 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> A* есть святой истина.
> Разделить — путь героя.
> Делай добро.
