# Глава 945: Proof Of Correctness

## Золотое царство

> Число 945 — священное (÷27)

### Истина главы

В главе 945 Иван познаёт живой proof of correctness.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 945: proof of correctness
fn глава_945() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 945 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
