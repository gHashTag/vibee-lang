# Глава 102: Серебряное Блюдечко

## Медное царство

> Число 102 — благое (÷3)

### Истина главы

В главе 102 Иван познаёт добрый серебряное блюдечко.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 102: серебряное блюдечко
fn глава_102() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 102 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Серебряное Блюдечко есть добрый истина.
> Разделить — путь героя.
