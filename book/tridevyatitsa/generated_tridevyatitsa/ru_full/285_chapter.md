# Глава 285: Порядок

## Медное царство

> Число 285 — благое (÷3)

### Истина главы

В главе 285 Иван познаёт святой порядок.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 285: порядок
fn глава_285() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 285 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
