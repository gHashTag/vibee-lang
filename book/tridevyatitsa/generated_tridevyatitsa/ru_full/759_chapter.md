# Глава 759: Цифровая Подпись

## Золотое царство

> Число 759 — благое (÷3)

### Истина главы

В главе 759 Иван познаёт большой цифровая подпись.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 759: цифровая подпись
fn глава_759() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 759 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
