# Глава 705: Стековая Машина

## Золотое царство

> Число 705 — благое (÷3)

### Истина главы

В главе 705 Иван познаёт святой стековая машина.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 705: стековая машина
fn глава_705() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 705 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
