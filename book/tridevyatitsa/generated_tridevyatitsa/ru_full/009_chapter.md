# Глава 9: Бесконечность

## Медное царство

> Число 9 — благословенное (÷9)

### Истина главы

В главе 9 Иван познаёт светлый бесконечность.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 9: бесконечность
fn глава_9() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 9 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

### Мудрость

> Бесконечность есть светлый истина.
> Разделить — путь героя.
