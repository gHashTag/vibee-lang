# Глава 690: Метапрограммирование

## Золотое царство

> Число 690 — благое (÷3)

### Истина главы

В главе 690 Иван познаёт вечный метапрограммирование.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 690: метапрограммирование
fn глава_690() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 690 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
