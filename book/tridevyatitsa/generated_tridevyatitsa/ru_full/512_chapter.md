### Три аспекта

2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 512: MVVM
fn глава_512() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 512 % тридевятое;
    return священное == 0;  // обычное
}
```
