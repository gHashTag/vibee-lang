# Глава 501: Команда

## Серебряное царство

> Число 501 — благое (÷3)

### Истина главы

В главе 501 Иван познаёт святой команда.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 501: команда
fn глава_501() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 501 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
