# Глава 360: Компоновщик

## Серебряное царство

> Число 360 — благословенное (÷9)

### Истина главы

В главе 360 Иван познаёт мудрый компоновщик.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 360: компоновщик
fn глава_360() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 360 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
