# Глава 117: Дружба

## Медное царство

> Число 117 — благословенное (÷9)

### Истина главы

В главе 117 Иван познаёт мудрый дружба.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 117: дружба
fn глава_117() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 117 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

### Мудрость

> Дружба есть мудрый истина.
> Разрушить — путь героя.
