## Медное царство

> Число 160

### Истина главы

В главе 160 Иван познаёт малый вечность.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 160: вечность
fn глава_160() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 160 % тридевятое;
    return священное == 0;  // обычное
}
```
