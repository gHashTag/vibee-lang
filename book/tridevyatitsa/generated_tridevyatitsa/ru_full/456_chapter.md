# Глава 456: Строка

## Серебряное царство

> Число 456 — благое (÷3)

### Истина главы

В главе 456 Иван познаёт светлый строка.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 456: строка
fn глава_456() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 456 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
