# Глава 520: Строка

## Серебряное царство

> Число 520

### Истина главы

В главе 520 Иван познаёт большой строка.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 520: строка
fn глава_520() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 520 % тридевятое;
    return священное == 0;  // обычное
}
```
