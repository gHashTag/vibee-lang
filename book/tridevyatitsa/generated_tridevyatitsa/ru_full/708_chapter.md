# Глава 708: Подсчёт Ссылок

## Золотое царство

> Число 708 — благое (÷3)

### Истина главы

В главе 708 Иван познаёт малый подсчёт ссылок.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 708: подсчёт ссылок
fn глава_708() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 708 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

> Подсчёт Ссылок есть малый истина.
> Разделить — путь героя.
