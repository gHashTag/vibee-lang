# Глава 77: Заколдованный Лес

## Медное царство

> Число 77

### Истина главы

В главе 77 Иван познаёт святой заколдованный лес.

### Три аспекта

1. **Первый аспект**: святой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: святой конец

### Код

```vibee
// Глава 77: заколдованный лес
fn глава_77() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 77 % тридевятое;
    return священное == 0;  // обычное
}
```
