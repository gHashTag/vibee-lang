# Глава 27: Полнота

## Медное царство

> Число 27 — священное (÷27)

### Истина главы

В главе 27 Иван познаёт светлый полнота.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 27: полнота
fn глава_27() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 27 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

### Мудрость

> Полнота есть светлый истина.
> Соединить — путь героя.
> Делай добро.
