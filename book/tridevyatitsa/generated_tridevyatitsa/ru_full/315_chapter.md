# Глава 315: Морозко

## Медное царство

> Число 315 — благословенное (÷9)

### Истина главы

В главе 315 Иван познаёт живой Морозко.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: найти середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 315: Морозко
fn глава_315() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 315 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
