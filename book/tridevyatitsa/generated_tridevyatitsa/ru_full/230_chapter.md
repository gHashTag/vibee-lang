# Глава 230: Живая Вода

## Медное царство

> Число 230

### Истина главы

В главе 230 Иван познаёт добрый живая вода.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 230: живая вода
fn глава_230() {

    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 230 % тридевятое;
    return священное == 0;  // обычное
}
```
