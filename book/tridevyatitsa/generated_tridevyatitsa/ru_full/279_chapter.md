# Глава 279: Бесконечность

## Медное царство

> Число 279 — благословенное (÷9)

### Истина главы

В главе 279 Иван познаёт живой бесконечность.

### Три аспекта

1. **Первый аспект**: живой начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: живой конец

### Код

```vibee
// Глава 279: бесконечность
fn глава_279() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 279 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```
