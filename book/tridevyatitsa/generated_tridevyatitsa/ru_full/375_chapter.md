# Глава 375: Интерпретатор

## Серебряное царство

> Число 375 — благое (÷3)

### Истина главы

В главе 375 Иван познаёт мудрый интерпретатор.

### Три аспекта

1. **Первый аспект**: мудрый начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: мудрый конец

### Код

```vibee
// Глава 375: интерпретатор
fn глава_375() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 375 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
