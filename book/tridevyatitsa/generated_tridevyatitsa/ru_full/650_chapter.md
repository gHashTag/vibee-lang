# Глава 650: Вещественное Число

## Серебряное царство

> Число 650

### Истина главы

В главе 650 Иван познаёт вечный вещественное число.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 650: вещественное число
fn глава_650() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 650 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Вещественное Число есть вечный истина.
> Разрушить — путь героя.
> Делай добро.
