# Глава 451: Список Смежности

## Серебряное царство

> Число 451

### Истина главы

В главе 451 Иван познаёт большой список смежности.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 451: список смежности
fn глава_451() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 451 % тридевятое;
    return священное == 0;  // обычное
}
```

### Мудрость

> Список Смежности есть большой истина.
> Разрушить — путь героя.
