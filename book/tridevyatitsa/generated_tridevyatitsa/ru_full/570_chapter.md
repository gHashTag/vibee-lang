# Глава 570: Двусвязный Список

## Серебряное царство

> Число 570 — благое (÷3)

### Истина главы

В главе 570 Иван познаёт малый двусвязный список.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 570: двусвязный список
fn глава_570() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 570 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
