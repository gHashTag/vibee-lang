# Глава 876: Аллокатор Памяти

## Золотое царство

> Число 876 — благое (÷3)

### Истина главы

В главе 876 Иван познаёт большой аллокатор памяти.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 876: аллокатор памяти
fn глава_876() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 876 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```
