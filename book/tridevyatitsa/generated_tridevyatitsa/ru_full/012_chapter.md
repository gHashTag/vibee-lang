# Глава 12: Свет

## Медное царство

> Число 12 — благое (÷3)

### Истина главы

В главе 12 Иван познаёт светлый свет.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 12: свет
fn глава_12() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 12 % тридевятое;
    return священное == 0;  // благое (÷3)
}
```

### Мудрость

> Свет есть светлый истина.
> Соединить — путь героя.
> Делай добро.
