# Глава 54: Гамаюн

## Медное царство

> Число 54 — священное (÷27)

### Истина главы

В главе 54 Иван познаёт добрый Гамаюн.

### Три аспекта

1. **Первый аспект**: добрый начало
2. **Второй аспект**: скрыть середину  
3. **Третий аспект**: добрый конец

### Код

```vibee
// Глава 54: Гамаюн
fn глава_54() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 54 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```
