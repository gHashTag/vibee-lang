# Глава 162: Полнота

## Медное царство

> Число 162 — священное (÷27)

### Истина главы

В главе 162 Иван познаёт большой полнота.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: преобразить середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 162: полнота
fn глава_162() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 162 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

> Полнота есть большой истина.
