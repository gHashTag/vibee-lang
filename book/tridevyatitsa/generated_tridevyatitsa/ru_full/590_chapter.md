# Глава 590: Кортеж

## Серебряное царство

> Число 590

### Истина главы

В главе 590 Иван познаёт большой кортеж.

### Три аспекта

1. **Первый аспект**: большой начало
2. **Второй аспект**: познать середину  
3. **Третий аспект**: большой конец

### Код

```vibee
// Глава 590: кортеж
fn глава_590() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 590 % тридевятое;
    return священное == 0;  // обычное
}
```
