# Глава 72: Хрустальный Дворец

## Медное царство

> Число 72 — благословенное (÷9)

### Истина главы

В главе 72 Иван познаёт малый хрустальный дворец.

### Три аспекта

1. **Первый аспект**: малый начало
2. **Второй аспект**: разрушить середину  
3. **Третий аспект**: малый конец

### Код

```vibee
// Глава 72: хрустальный дворец
fn глава_72() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 72 % тридевятое;
    return священное == 0;  // благословенное (÷9)
}
```

### Мудрость

> Хрустальный Дворец есть малый истина.
> Разрушить — путь героя.
> Делай добро.
