# Глава 74: Подводное Царство

## Медное царство

> Число 74

### Истина главы

В главе 74 Иван познаёт тёмный подводное царство.

### Три аспекта

1. **Первый аспект**: тёмный начало
2. **Второй аспект**: создать середину  
3. **Третий аспект**: тёмный конец

### Код

```vibee
// Глава 74: подводное царство
fn глава_74() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 74 % тридевятое;
    return священное == 0;  // обычное
}
```
