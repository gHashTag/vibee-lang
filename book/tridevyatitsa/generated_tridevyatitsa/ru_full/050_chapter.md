# Глава 50: Русалка

## Медное царство

> Число 50

### Истина главы

В главе 50 Иван познаёт светлый Русалка.

### Три аспекта

1. **Первый аспект**: светлый начало
2. **Второй аспект**: соединить середину  
3. **Третий аспект**: светлый конец

### Код

```vibee
// Глава 50: Русалка
fn глава_50() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 50 % тридевятое;
    return священное == 0;  // обычное
}
```
