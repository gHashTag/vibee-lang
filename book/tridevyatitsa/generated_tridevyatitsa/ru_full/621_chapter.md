# Глава 621: Поиск Точек Сочленения

## Серебряное царство

> Число 621 — священное (÷27)

### Истина главы

В главе 621 Иван познаёт вечный поиск точек сочленения.

### Три аспекта

1. **Первый аспект**: вечный начало
2. **Второй аспект**: разделить середину  
3. **Третий аспект**: вечный конец

### Код

```vibee
// Глава 621: поиск точек сочленения
fn глава_621() {
    let тройка = 3;
    let тридевятое = тройка * тройка * тройка;  // 27
    let священное = 621 % тридевятое;
    return священное == 0;  // священное (÷27)
}
```

### Мудрость

> Поиск Точек Сочленения есть вечный истина.
> Разделить — путь героя.
