# Книга 1: Начало Пути — База знаний

## Научное содержание

### История троичных систем

**1840** — Томас Фаулер построил первый троичный калькулятор из дерева и проволоки в Англии. Он обнаружил, что троичная система требует меньше деталей для представления чисел.

**1958** — Николай Брусенцов в МГУ создал **Сетунь** — первый и единственный серийный троичный компьютер. Было выпущено 50 машин. Сетунь использовала **сбалансированную троичную систему** (-1, 0, +1).

**Преимущества троичной системы:**
- Меньше разрядов для представления чисел (log₃ n < log₂ n)
- Естественное представление отрицательных чисел
- Округление без смещения
- Меньше операций переноса при сложении

### Математика числа 3

**Теорема о единственности разложения:**
Любое натуральное число N единственным образом представляется в виде N = n × 3^k, где n не делится на 3.

**Доказательство:**
1. Существование: делим N на 3, пока делится
2. Единственность: от противного — если N = n₁ × 3^k₁ = n₂ × 3^k₂, то n₁ = n₂ и k₁ = k₂

**Связь с золотым сечением:**
φ² + 1/φ² = 3 (точно!)

Это фундаментальное тождество связывает золотое сечение с числом 3.

### Три системы восприятия

По исследованиям когнитивной науки, человек воспринимает информацию через три канала:

1. **Интуиция (20%)** — образное мышление, метафоры, аналогии
2. **Анализ (60%)** — логическое мышление, формулы, доказательства
3. **Синтез (20%)** — интеграция, инсайты, мудрость

Эффективное обучение задействует все три системы.

### Пример 3: Золотое тождество

```999
// Проверка тождества φ² + 1/φ² = 3
ⲙⲟⲇⲩⲗⲉ ⲍⲟⲗⲟⲧⲟⲉ_ⲧⲟⲍⲇⲉⲥⲧⲃⲟ;

ⲕⲟⲛⲥⲧ φ: f64 = 1.6180339887498948482;

ⲫⲩⲛⲕ проверить_тождество() -> bool {
    ⲕⲟⲛⲥⲧ φ_квадрат = φ * φ;           // ≈ 2.618
    ⲕⲟⲛⲥⲧ обратный_квадрат = 1.0 / φ_квадрат;  // ≈ 0.382
    ⲕⲟⲛⲥⲧ сумма = φ_квадрат + обратный_квадрат;
    
    // Должно быть ровно 3
    ⲣⲉⲧⲩⲣⲛ @abs(сумма - 3.0) < 1e-15;
}

ⲫⲩⲛⲕ main() !void {
    ⲡⲣⲓⲛⲧ("φ = {d:.20}", φ);
    ⲡⲣⲓⲛⲧ("φ² = {d:.20}", φ * φ);
    ⲡⲣⲓⲛⲧ("1/φ² = {d:.20}", 1.0 / (φ * φ));
    ⲡⲣⲓⲛⲧ("φ² + 1/φ² = {d:.20}", φ * φ + 1.0 / (φ * φ));
    
    ⲓⲫ (проверить_тождество()) {
        ⲡⲣⲓⲛⲧ("✓ Тождество подтверждено: φ² + 1/φ² = 3");
    }
}
```
